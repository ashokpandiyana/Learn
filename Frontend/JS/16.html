‚≠ê 2. Generators

2.1 Generator Functions

// Generator function (function*) function* numberGenerator() { yield 1; yield 2; yield 3; } const gen = numberGenerator(); console.log(gen.next()); // { value: 1, done: false } console.log(gen.next()); // { value: 2, done: false } console.log(gen.next()); // { value: 3, done: false } console.log(gen.next()); // { done: true } // Infinite sequences function* infiniteNumbers() { let i = 0; while (true) { yield i++; } } const nums = infiniteNumbers(); console.log(nums.next().value); // 0 console.log(nums.next().value); // 1

2.2 yield* Delegation

function* gen1() { yield 1; yield 2; } function* gen2() { yield* gen1(); // Delegate to gen1 yield 3; } console.log([...gen2()]); // [1, 2, 3]

üìù Summary

Key Takeaways:
  • Iterators define how objects are iterated
  • Generators are pausable functions (function*)
  • yield pauses execution, returns value
  • yield* delegates to another generator
  • Great for lazy evaluation & infinite sequences
