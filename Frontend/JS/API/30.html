Drawing Graphics with JavaScript

üéØ What is the Canvas API?

Canvas API provides a means to draw graphics via JavaScript using the HTML <canvas> element. It's a powerful tool for creating charts, games, animations, image manipulation, and more.

Key Features:

  • üé® 2D Drawing: Shapes, paths, text, images
  • üñºÔ∏è Pixel Manipulation: Direct pixel access
  • ‚ö° Fast: Hardware-accelerated rendering
  • üé¨ Animations: Frame-by-frame updates
  • üìä Data Visualization: Charts and graphs
  • üéÆ Games: 2D game graphics
üí° Common Use Cases:
  • Charts and data visualization
  • Image editing and filters
  • 2D games
  • Signature pads
  • Diagrams and flowcharts
  • Animations and effects

üöÄ Getting Started

Creating a Canvas

// HTML // <canvas id="myCanvas" width="800" height="600"></canvas> // Get canvas and context const canvas = document.getElementById('myCanvas'); const ctx = canvas.getContext('2d'); // Set canvas size canvas.width = 800; canvas.height = 600; // Get canvas dimensions console.log(canvas.width, canvas.height);
‚ö†Ô∏è Important: Set canvas size via width and height attributes/properties, NOT CSS! CSS scales the canvas but doesn't change resolution, causing blurry graphics.

üìê Drawing Shapes

Rectangles

// Filled rectangle ctx.fillStyle = 'red'; ctx.fillRect(x, y, width, height); // Stroked (outline) rectangle ctx.strokeStyle = 'blue'; ctx.lineWidth = 3; ctx.strokeRect(x, y, width, height); // Clear rectangle area ctx.clearRect(x, y, width, height); // Example: Draw squares ctx.fillStyle = '#667eea'; ctx.fillRect(50, 50, 100, 100); ctx.strokeStyle = '#764ba2'; ctx.lineWidth = 5; ctx.strokeRect(200, 50, 100, 100);

Paths (Lines, Circles, Custom Shapes)

// Drawing a path ctx.beginPath(); // Start new path ctx.moveTo(x, y); // Move to start point ctx.lineTo(x, y); // Draw line to point ctx.closePath(); // Close path (optional) ctx.stroke(); // Draw outline ctx.fill(); // Fill shape // Example: Triangle ctx.beginPath(); ctx.moveTo(100, 50); ctx.lineTo(150, 150); ctx.lineTo(50, 150); ctx.closePath(); ctx.fillStyle = 'orange'; ctx.fill(); // Circle (using arc) ctx.beginPath(); ctx.arc(x, y, radius, startAngle, endAngle, counterclockwise); ctx.stroke(); // Example: Full circle ctx.beginPath(); ctx.arc(200, 200, 50, 0, Math.PI * 2); ctx.fillStyle = 'blue'; ctx.fill(); // Arc (partial circle) ctx.beginPath(); ctx.arc(300, 200, 50, 0, Math.PI); // Half circle ctx.stroke(); // Curves ctx.beginPath(); ctx.moveTo(50, 50); ctx.quadraticCurveTo(cpx, cpy, x, y); // Quadratic curve ctx.bezierCurveTo(cp1x, cp1y, cp2x, cp2y, x, y); // Bezier curve ctx.stroke();

üé® Styling

// Fill color ctx.fillStyle = 'red'; ctx.fillStyle = '#ff0000'; ctx.fillStyle = 'rgb(255, 0, 0)'; ctx.fillStyle = 'rgba(255, 0, 0, 0.5)'; // Stroke color ctx.strokeStyle = 'blue'; // Line width ctx.lineWidth = 5; // Line cap (end of lines) ctx.lineCap = 'butt'; // 'butt', 'round', 'square' // Line join (corner style) ctx.lineJoin = 'miter'; // 'miter', 'round', 'bevel' // Dashed lines ctx.setLineDash([5, 10]); // [dash length, gap length] ctx.setLineDash([]); // Reset to solid // Transparency ctx.globalAlpha = 0.5; // 0.0 to 1.0 // Gradients const gradient = ctx.createLinearGradient(0, 0, 200, 0); gradient.addColorStop(0, 'red'); gradient.addColorStop(0.5, 'yellow'); gradient.addColorStop(1, 'blue'); ctx.fillStyle = gradient; ctx.fillRect(0, 0, 200, 100); // Radial gradient const radial = ctx.createRadialGradient(100, 100, 10, 100, 100, 50); radial.addColorStop(0, 'white'); radial.addColorStop(1, 'blue'); ctx.fillStyle = radial;

‚úèÔ∏è Drawing Text

// Set font ctx.font = '48px serif'; ctx.font = 'bold 24px Arial'; ctx.font = 'italic 20px Georgia'; // Text alignment ctx.textAlign = 'start'; // 'start', 'end', 'left', 'right', 'center' // Text baseline ctx.textBaseline = 'top'; // 'top', 'hanging', 'middle', 'alphabetic', 'bottom' // Draw filled text ctx.fillStyle = 'black'; ctx.fillText('Hello Canvas!', x, y); ctx.fillText('Text', x, y, maxWidth); // With max width // Draw stroked text ctx.strokeStyle = 'blue'; ctx.lineWidth = 2; ctx.strokeText('Outlined', x, y); // Measure text const metrics = ctx.measureText('Hello'); console.log('Width:', metrics.width);

üñºÔ∏è Drawing Images

// Load image const img = new Image(); img.onload = () => { // Draw when loaded ctx.drawImage(img, x, y); }; img.src = 'image.png'; // Draw with size ctx.drawImage(img, x, y, width, height); // Draw portion of image (sprite sheet) ctx.drawImage( img, sx, sy, sWidth, sHeight, // Source rectangle dx, dy, dWidth, dHeight // Destination rectangle ); // Example: Draw and resize const image = new Image(); image.onload = () => { ctx.drawImage(image, 0, 0, 200, 150); }; image.src = 'photo.jpg';

üîÑ Transformations

// Translate (move origin) ctx.translate(x, y); // Rotate (in radians) ctx.rotate(angle); ctx.rotate(Math.PI / 4); // 45 degrees // Scale ctx.scale(scaleX, scaleY); ctx.scale(2, 2); // Double size ctx.scale(-1, 1); // Flip horizontal // Save and restore state ctx.save(); // Save current state ctx.translate(100, 100); ctx.rotate(Math.PI / 4); ctx.fillRect(0, 0, 50, 50); ctx.restore(); // Restore to saved state // Example: Rotate around center ctx.save(); ctx.translate(200, 200); // Move to center ctx.rotate(Math.PI / 4); // Rotate ctx.fillRect(-25, -25, 50, 50); // Draw centered ctx.restore();

‚úÖ Best Practices

DO ‚úì

  • Use save() and restore() for transformations
  • Clear canvas before redrawing animations
  • Use requestAnimationFrame for smooth animations
  • Optimize by redrawing only changed areas
  • Set canvas size via attributes, not CSS
