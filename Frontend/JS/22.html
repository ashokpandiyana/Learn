üåç 1. Fetch API

// Modern HTTP requests const response = await fetch('/api/users'); const data = await response.json(); // POST request await fetch('/api/users', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ name: "Alice" }) }); // With error handling try { const response = await fetch('/api/users'); if (!response.ok) { throw new Error(`HTTP ${response.status}`); } const data = await response.json(); } catch (error) { console.error('Fetch failed:', error); }

üíæ 2. Storage APIs

2.1 LocalStorage

// Store data persistently (5-10MB) localStorage.setItem('user', JSON.stringify({ name: "Alice" })); const user = JSON.parse(localStorage.getItem('user')); localStorage.removeItem('user'); localStorage.clear();

2.2 SessionStorage

// Same API, but clears when tab closes sessionStorage.setItem('temp', 'data');

2.3 IndexedDB

// Large-scale client-side storage const request = indexedDB.open('MyDatabase', 1); request.onsuccess = () => { const db = request.result; // Use database... };

üîî 3. Observers

3.1 IntersectionObserver

// Detect when elements enter viewport const observer = new IntersectionObserver(entries => { entries.forEach(entry => { if (entry.isIntersecting) { console.log('Element is visible!'); // Lazy load image, trigger animation, etc. } }); }); observer.observe(document.querySelector('.lazy-image'));

3.2 MutationObserver

// Watch for DOM changes const observer = new MutationObserver(mutations => { mutations.forEach(mutation => { console.log('DOM changed:', mutation); }); }); observer.observe(document.body, { childList: true, subtree: true });

üìù Summary

Essential Browser APIs:
  • Fetch - HTTP requests
  • Storage - localStorage, sessionStorage, IndexedDB
  • Observers - Intersection, Mutation, Resize
  • Service Workers - Offline functionality, PWA
  • Geolocation - User location
  • Notifications - Push notifications
  • WebSockets - Real-time communication
