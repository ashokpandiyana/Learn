üîÑ 1. flat() and flatMap()

1.1 flat() - Flatten Nested Arrays

// Basic flattening (depth 1) const nested = [1, [2, 3], [4, [5]]]; console.log(nested.flat()); // [1, 2, 3, 4, [5]] // Flatten to depth 2 console.log(nested.flat(2)); // [1, 2, 3, 4, 5] // Flatten completely (Infinity) const deepNested = [1, [2, [3, [4, [5]]]]]; console.log(deepNested.flat(Infinity)); // [1, 2, 3, 4, 5] // Remove empty slots const sparse = [1, , , 2, , 3]; console.log(sparse.flat()); // [1, 2, 3] (removes holes)

1.2 flatMap() - Map then Flatten

// flatMap = map + flat(1) const words = ["hello world", "foo bar"]; // With map + flat const result1 = words.map(s => s.split(" ")).flat(); console.log(result1); // ["hello", "world", "foo", "bar"] // With flatMap (cleaner) const result2 = words.flatMap(s => s.split(" ")); console.log(result2); // ["hello", "world", "foo", "bar"] // Practical: Extract all tags from posts const posts = [ { title: "Post 1", tags: ["js", "react"] }, { title: "Post 2", tags: ["node", "js"] } ]; const allTags = posts.flatMap(post => post.tags); console.log(allTags); // ["js", "react", "node", "js"]

// Like reduce, but iterates right-to-left const numbers = [1, 2, 3, 4]; // reduce (left to right) const result1 = numbers.reduce((acc, n) => `(${acc}+${n})`, "0"); console.log(result1); // "(((0+1)+2)+3)+4)" // reduceRight (right to left) const result2 = numbers.reduceRight((acc, n) => `(${acc}+${n})`, "0"); console.log(result2); // "(((0+4)+3)+2)+1)" // Practical: Function composition const compose = (...fns) => x => fns.reduceRight((value, fn) => fn(value), x); // Practical: Reverse build const buildPath = parts.reduceRight((path, part) => part + "/" + path );

'A' : 'B' })) .sort((a, b) => b.score - a.score) // Sort by score descending .map(u => u.name); // Extract names console.log(result); // ["David", "Alice", "Charlie"]

üõ†Ô∏è 4. Custom Array Method Implementations

4.1 Implement map()

// Polyfill for Array.prototype.map Array.prototype.myMap = function(callback) { const result = []; for (let i = 0; i < this.length; i++) { result.push(callback(this[i], i, this)); } return result; }; // Test console.log([1, 2, 3].myMap(x => x * 2)); // [2, 4, 6]

4.2 Implement filter()

Array.prototype.myFilter = function(callback) { const result = []; for (let i = 0; i < this.length; i++) { if (callback(this[i], i, this)) { result.push(this[i]); } } return result; };

4.3 Implement reduce()

Array.prototype.myReduce = function(callback, initialValue) { let accumulator = initialValue !== undefined ? initialValue : this[0]; let startIndex = initialValue !== undefined ? 0 : 1; for (let i = startIndex; i < this.length; i++) { accumulator = callback(accumulator, this[i], i, this); } return accumulator; }; console.log([1, 2, 3, 4].myReduce((a, b) => a + b)); // 10

üî¢ 5. Sorting with Custom Comparators

// Default sort (converts to strings!) const numbers = [1, 30, 4, 21, 100000]; console.log(numbers.sort()); // [1, 100000, 21, 30, 4] ‚ùå Wrong! // Numeric sort (ascending) numbers.sort((a, b) => a - b); console.log(numbers); // [1, 4, 21, 30, 100000] ‚úÖ // Descending numbers.sort((a, b) => b - a); // Sort objects const users = [ { name: "Charlie", age: 30 }, { name: "Alice", age: 25 }, { name: "Bob", age: 25 } ]; // Sort by age, then name users.sort((a, b) => { if (a.age !== b.age) { return a.age - b.age; } return a.name.localeCompare(b.name); });

üìù Summary

Key Takeaways:
  • flat() flattens nested arrays to specified depth
  • flatMap() = map() + flat(1) in one operation
  • reduceRight() processes array right-to-left
  • Method chaining creates powerful data pipelines
  • Custom implementations help understand internals
  • sort() requires comparator for numbers
  • Array-like objects can be converted to arrays
