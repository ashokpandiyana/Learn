üéØ Key Takeaways

  • ‚úì Combinators: Space (any), > (direct), + (next), ~ (all after)
  • ‚úì :nth-child(): Use formulas like 2n (even), 2n+1 (odd), 3n+1, -n+3 (first 3)
  • ‚úì :has(): Parent selector! Selects elements containing specific children
  • ‚úì Pseudo-elements: ::before, ::after (require content), ::first-letter, ::selection
  • ‚úì Specificity: Inline (1,0,0,0) > ID (0,1,0,0) > Class (0,0,1,0) > Element (0,0,0,1)
  • ‚úì LVHA order: :link, :visited, :hover, :active (for links)
