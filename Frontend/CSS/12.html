1. Separate structure from skin

üéØ Key Takeaways

  • ‚úì BEM: Block__Element--Modifier - Most popular methodology
  • ‚úì Design Tokens: Single source of truth for design values (colors, spacing, typography)
  • ‚úì Keep Specificity Low: Use classes, avoid IDs and deep nesting
  • ‚úì Component Isolation: Components should work anywhere, not depend on parents
  • ‚úì File Organization: Group by component/feature, not by type
  • ‚úì Consistency: Choose ONE methodology and stick to it across the project
