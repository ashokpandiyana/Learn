# Chapter 5: Flexbox - In-Depth Explanation

## Introduction to Flexbox

**Flexbox (Flexible Box Layout)** is a one-dimensional layout system designed for distributing space and aligning items within a container, even when their size is unknown or dynamic.

**Key Concept:** Flexbox operates on a **main axis** and **cross axis**.

```
flex-direction: row (default)
┌─────────────────────────────────┐
│  Main Axis →                    │
│  ┌────┐ ┌────┐ ┌────┐          │
│  │ 1  │ │ 2  │ │ 3  │  ↕       │
│  └────┘ └────┘ └────┘  Cross   │
│                         Axis    │
└─────────────────────────────────┘

flex-direction: column
┌──────────────┐
│  Cross Axis  │
│      ↔       │
│  ┌────────┐  │
│  │   1    │  │  ↓
│  └────────┘  │  Main
│  ┌────────┐  │  Axis
│  │   2    │  │  ↓
│  └────────┘  │
│  ┌────────┐  │
│  │   3    │  │
│  └────────┘  │
└──────────────┘
```

**Important:** Understanding main axis vs cross axis is CRITICAL for Flexbox!

## 5.1 Flex Container Properties

### display: flex

Creates a flex container and makes direct children flex items.

```css
.container {
    display: flex;        /* Block-level flex container */
    display: inline-flex; /* Inline-level flex container */
}
```

**What happens:**
- Container becomes flex container
- **Direct children** become flex items (grandchildren are NOT flex items)
- Items layout according to flex rules
- Block formatting context is created

**Example:**
```html
<div class="flex-container">
    <div>Item 1</div>     <!-- Flex item -->
    <div>Item 2</div>     <!-- Flex item -->
    <div>
        Item 3
        <span>Child</span> <!-- NOT a flex item -->
    </div>
</div>
```

```css
.flex-container {
    display: flex;
}
/* Only direct children (Item 1, 2, 3) are flex items */
/* span inside Item 3 is NOT a flex item */
```

---

### flex-direction

Defines the **main axis** direction (how items are laid out).

```css
.container {
    flex-direction: row;            /* Default: left to right */
    flex-direction: row-reverse;    /* Right to left */
    flex-direction: column;         /* Top to bottom */
    flex-direction: column-reverse; /* Bottom to top */
}
```

**Visual Representation:**

**row (default):**
```
┌────────────────────────────┐
│  [1] [2] [3] [4]  →        │  Main axis
└────────────────────────────┘
```

**row-reverse:**
```
┌────────────────────────────┐
│        ← [4] [3] [2] [1]   │  Main axis (reversed)
└────────────────────────────┘
```

**column:**
```
┌──────┐
│ [1]  │  ↓
│ [2]  │  Main
│ [3]  │  axis
│ [4]  │  ↓
└──────┘
```

**column-reverse:**
```
┌──────┐
│ [4]  │  ↑
│ [3]  │  Main
│ [2]  │  axis
│ [1]  │  ↑
└──────┘
```

**Examples:**
```css
/* Horizontal menu */
.nav {
    display: flex;
    flex-direction: row;
}

/* Vertical sidebar */
.sidebar {
    display: flex;
    flex-direction: column;
}

/* Reverse order for RTL languages */
.rtl-menu {
    display: flex;
    flex-direction: row-reverse;
}
```

---

### flex-wrap

Controls whether items wrap to new lines when they don't fit.

```css
.container {
    flex-wrap: nowrap;      /* Default: single line, may overflow */
    flex-wrap: wrap;        /* Wrap to multiple lines */
    flex-wrap: wrap-reverse; /* Wrap, but reversed */
}
```

**nowrap (default):**
```
┌────────────────────────────┐
│ [1] [2] [3] [4] [5] [6] →  │  All items squeezed on one line
└────────────────────────────┘
```

**wrap:**
```
┌────────────────────────────┐
│ [1] [2] [3] [4]            │  ← Line 1
│ [5] [6]                    │  ← Line 2
└────────────────────────────┘
```

**wrap-reverse:**
```
┌────────────────────────────┐
│ [5] [6]                    │  ← Line 2 (appears first!)
│ [1] [2] [3] [4]            │  ← Line 1
└────────────────────────────┘
```

**Examples:**
```css
/* Responsive grid */
.gallery {
    display: flex;
    flex-wrap: wrap;
}
.gallery-item {
    width: 200px;  /* Items wrap when container too narrow */
}

/* Single-line navigation (overflow if needed) */
.nav {
    display: flex;
    flex-wrap: nowrap;
    overflow-x: auto;  /* Scroll if too many items */
}
```

---

### flex-flow (Shorthand)

Combines `flex-direction` and `flex-wrap`.

```css
/* Longhand */
.container {
    flex-direction: row;
    flex-wrap: wrap;
}

/* Shorthand */
.container {
    flex-flow: row wrap;
}

/* Other examples */
.container {
    flex-flow: column nowrap;
    flex-flow: row-reverse wrap;
    flex-flow: column wrap-reverse;
}
```

---

### justify-content

Aligns items along the **main axis** (distributes free space).

```css
.container {
    justify-content: flex-start;    /* Default: start of main axis */
    justify-content: flex-end;      /* End of main axis */
    justify-content: center;        /* Center of main axis */
    justify-content: space-between; /* First/last at edges, equal space between */
    justify-content: space-around;  /* Equal space around each item */
    justify-content: space-evenly;  /* Equal space everywhere */
}
```

**Visual Comparison (flex-direction: row):**

**flex-start (default):**
```
┌────────────────────────────────────┐
│ [1][2][3]                          │  ← Items at start
└────────────────────────────────────┘
```

**flex-end:**
```
┌────────────────────────────────────┐
│                          [1][2][3] │  ← Items at end
└────────────────────────────────────┘
```

**center:**
```
┌────────────────────────────────────┐
│            [1][2][3]               │  ← Items centered
└────────────────────────────────────┘
```

**space-between:**
```
┌────────────────────────────────────┐
│ [1]         [2]         [3]        │  ← First/last at edges
└────────────────────────────────────┘
```

**space-around:**
```
┌────────────────────────────────────┐
│   [1]       [2]       [3]          │  ← Equal space around
└────────────────────────────────────┘
    ↑         ↑         ↑
  1 unit    2 units   2 units
  (edges have half the space)
```

**space-evenly:**
```
┌────────────────────────────────────┐
│    [1]      [2]      [3]           │  ← Equal space everywhere
└────────────────────────────────────┘
     ↑        ↑        ↑
   Same     Same     Same
```

**Examples:**
```css
/* Center button group */
.button-group {
    display: flex;
    justify-content: center;
}

/* Navigation with space between */
.nav {
    display: flex;
    justify-content: space-between;
}

/* Icon group at end of header */
.header-icons {
    display: flex;
    justify-content: flex-end;
}

/* Even spacing for cards */
.card-container {
    display: flex;
    justify-content: space-evenly;
}
```

### align-items

Aligns items along the **cross axis** (perpendicular to main axis).

```css
.container {
    align-items: stretch;      /* Default: stretch to fill container */
    align-items: flex-start;   /* Start of cross axis */
    align-items: flex-end;     /* End of cross axis */
    align-items: center;       /* Center of cross axis */
    align-items: baseline;     /* Align text baselines */
}
```

**Visual Comparison (flex-direction: row):**

**stretch (default):**
```
┌────────────────────────────────────┐
│ ┌────┐ ┌────┐ ┌────┐              │  ← All items stretch
│ │ 1  │ │ 2  │ │ 3  │              │     to container height
│ │    │ │    │ │    │              │
│ └────┘ └────┘ └────┘              │
└────────────────────────────────────┘
```

**flex-start:**
```
┌────────────────────────────────────┐
│ ┌──┐ ┌──┐ ┌──┐                    │  ← Items at top
│ │1 │ │2 │ │3 │                    │     (start of cross axis)
│ └──┘ └──┘ └──┘                    │
│                                    │
└────────────────────────────────────┘
```

**flex-end:**
```
┌────────────────────────────────────┐
│                                    │
│ ┌──┐ ┌──┐ ┌──┐                    │  ← Items at bottom
│ │1 │ │2 │ │3 │                    │     (end of cross axis)
│ └──┘ └──┘ └──┘                    │
└────────────────────────────────────┘
```

**center:**
```
┌────────────────────────────────────┐
│                                    │
│ ┌──┐ ┌──┐ ┌──┐                    │  ← Items vertically centered
│ │1 │ │2 │ │3 │                    │
│ └──┘ └──┘ └──┘                    │
│                                    │
└────────────────────────────────────┘
```

**baseline:**
```
┌────────────────────────────────────┐
│     ┌──────┐                       │
│ ┌─┐ │  2   │ ┌───┐                │  ← Text baselines aligned
│ │1│ │      │ │ 3 │                │
│ └─┘ └──────┘ └───┘                │
│     ════════════  ← Baseline       │
└────────────────────────────────────┘
```

**Examples:**
```css
/* Vertically center navigation items */
.nav {
    display: flex;
    align-items: center;
    height: 60px;
}

/* Align form labels with inputs */
.form-row {
    display: flex;
    align-items: baseline;
}

/* Cards with equal height */
.card-container {
    display: flex;
    align-items: stretch;  /* Cards stretch to match tallest */
}
```

### align-content

Aligns **flex lines** (when wrapping) along cross axis. **Only works with multiple lines!**

```css
.container {
    flex-wrap: wrap;  /* Required for align-content to work */
    align-content: stretch;        /* Default */
    align-content: flex-start;     /* Lines at start */
    align-content: flex-end;       /* Lines at end */
    align-content: center;         /* Lines centered */
    align-content: space-between;  /* Lines with space between */
    align-content: space-around;   /* Lines with space around */
    align-content: space-evenly;   /* Lines with even space */
}
```

**Important:** `align-content` vs `align-items`
- `align-items`: Aligns items **within each line**
- `align-content`: Aligns **entire lines** themselves

**Visual Comparison (with wrap):**

**flex-start:**
```
┌────────────────────────────────────┐
│ [1] [2] [3] [4]                    │  ← Line 1
│ [5] [6]                            │  ← Line 2
│                                    │
│                                    │
└────────────────────────────────────┘
```

**center:**
```
┌────────────────────────────────────┐
│                                    │
│ [1] [2] [3] [4]                    │  ← Line 1 (centered)
│ [5] [6]                            │  ← Line 2
│                                    │
└────────────────────────────────────┘
```

**space-between:**
```
┌────────────────────────────────────┐
│ [1] [2] [3] [4]                    │  ← Line 1 (at top)
│                                    │
│ [5] [6]                            │  ← Line 2 (at bottom)
└────────────────────────────────────┘
```

**Example:**
```css
/* Multi-row gallery with centered rows */
.gallery {
    display: flex;
    flex-wrap: wrap;
    align-content: center;
    height: 500px;
}
```

## 5.2 Flex Item Properties

### flex-grow

Controls how much an item **grows** relative to siblings when distributing **extra space**.

```css
.item {
    flex-grow: 0;  /* Default: don't grow */
    flex-grow: 1;  /* Grow proportionally */
    flex-grow: 2;  /* Grow 2x as much */
}
```

**How it works:**
- Default is 0 (items don't grow)
- Positive number = item can grow
- **Proportion** of available space, not absolute size

**Example:**
```css
.container {
    display: flex;
    width: 600px;
}
.item1 { flex-grow: 1; }  /* Gets 1/3 of extra space */
.item2 { flex-grow: 1; }  /* Gets 1/3 of extra space */
.item3 { flex-grow: 1; }  /* Gets 1/3 of extra space */
```

**Visual:**
```
Container width: 600px
Items base size: 100px each (300px total)
Extra space: 600 - 300 = 300px

flex-grow: 1 each (total: 3)
Each item gets: 300px ÷ 3 = 100px extra

┌─────────────────────────────────────────────────────────┐
│ ┌────────────────┐ ┌────────────────┐ ┌────────────────┐ │
│ │  Item 1 (200)  │ │  Item 2 (200)  │ │  Item 3 (200)  │ │
│ └────────────────┘ └────────────────┘ └────────────────┘ │
└─────────────────────────────────────────────────────────┘
```

**Different grow values:**
```css
.item1 { flex-grow: 1; }  /* Gets 1/4 of extra space */
.item2 { flex-grow: 2; }  /* Gets 2/4 (half) of extra space */
.item3 { flex-grow: 1; }  /* Gets 1/4 of extra space */
```

**Visual:**
```
Extra space: 300px
Total grow factor: 1 + 2 + 1 = 4

Item 1: 100 + (300 × 1/4) = 175px
Item 2: 100 + (300 × 2/4) = 250px
Item 3: 100 + (300 × 1/4) = 175px

┌─────────────────────────────────────────────────────────┐
│ ┌─────────┐ ┌──────────────┐ ┌─────────┐               │
│ │Item 1   │ │   Item 2     │ │Item 3   │               │
│ │(175px)  │ │   (250px)    │ │(175px)  │               │
│ └─────────┘ └──────────────┘ └─────────┘               │
└─────────────────────────────────────────────────────────┘
```

**Common Patterns:**
```css
/* Main content takes remaining space */
.sidebar { width: 250px; }
.main { flex-grow: 1; }  /* Fills rest */

/* Equal-width items */
.item { flex-grow: 1; }  /* All grow equally */

/* Featured item larger */
.item { flex-grow: 1; }
.featured { flex-grow: 2; }  /* 2x larger */
```

### flex-shrink

Controls how much an item **shrinks** when space is insufficient.

```css
.item {
    flex-shrink: 1;  /* Default: can shrink */
    flex-shrink: 0;  /* Don't shrink (stay at base size) */
    flex-shrink: 2;  /* Shrink 2x as much */
}
```

**How it works:**
- Default is 1 (items can shrink equally)
- 0 = item won't shrink (maintains its size)
- Higher number = shrinks more

**Example:**
```css
.container {
    display: flex;
    width: 400px;  /* Container too small */
}
.item { width: 200px; }  /* Each wants 200px (600px total) */
/* Need to shrink by 200px */
```

**With flex-shrink: 1 (default):**
```
Each shrinks by: 200px ÷ 3 = 66.67px
Final: 133.33px each

┌─────────────────────────────────────────────┐
│ ┌────────┐ ┌────────┐ ┌────────┐           │
│ │Item 1  │ │Item 2  │ │Item 3  │           │
│ │(133px) │ │(133px) │ │(133px) │           │
│ └────────┘ └────────┘ └────────┘           │
└─────────────────────────────────────────────┘
```

**Prevent shrinking:**
```css
.item1 { flex-shrink: 0; }  /* Won't shrink */
.item2 { flex-shrink: 1; }  /* Can shrink */
.item3 { flex-shrink: 1; }  /* Can shrink */
```

```
Item 1: 200px (doesn't shrink)
Items 2 & 3 share the deficit: 200px ÷ 2 = 100px each

┌─────────────────────────────────────────────┐
│ ┌─────────┐ ┌─────┐ ┌─────┐                │
│ │ Item 1  │ │Item │ │Item │                │
│ │ (200px) │ │ 2   │ │ 3   │                │
│ └─────────┘ │(100)│ │(100)│                │
│             └─────┘ └─────┘                │
└─────────────────────────────────────────────┘
```

**Common Patterns:**
```css
/* Prevent logo from shrinking */
.logo {
    width: 200px;
    flex-shrink: 0;  /* Always 200px */
}

/* Allow content to shrink */
.content {
    flex-shrink: 1;
}
```

### flex-basis

Sets the **initial size** of a flex item before growing/shrinking.

```css
.item {
    flex-basis: auto;     /* Default: use width/height */
    flex-basis: 200px;    /* Start at 200px */
    flex-basis: 50%;      /* 50% of container */
    flex-basis: 0;        /* No base size (pure grow/shrink) */
}
```

**flex-basis vs width:**
- `flex-basis` is the "ideal size" before flex algorithm
- Overrides `width` (in row) or `height` (in column)
- More flexible than width/height

**Examples:**
```css
/* Start at 200px, can grow */
.item {
    flex-basis: 200px;
    flex-grow: 1;
}

/* Percentage-based */
.sidebar {
    flex-basis: 25%;     /* Start at 25% of container */
    flex-grow: 0;        /* Don't grow */
    flex-shrink: 0;      /* Don't shrink */
}

/* Pure flex (no base size) */
.item {
    flex-basis: 0;       /* Start with no size */
    flex-grow: 1;        /* Size determined by flex-grow only */
}
```

**Understanding flex-basis: 0 vs auto:**

**flex-basis: auto** (default):
- Item starts at content size or width/height
- Then grows/shrinks from there

**flex-basis: 0:**
- Item starts with no size
- All space distributed by flex-grow
- Results in more "equal" sizes

```css
/* Example comparison */

/* With flex-basis: auto */
.item { flex: 1; }  /* flex: 1 1 auto */
/* Items with more content will be larger initially */

/* With flex-basis: 0 */
.item { flex: 1 1 0; }
/* All items truly equal regardless of content */
```

### flex Shorthand (IMPORTANT!)

Combines `flex-grow`, `flex-shrink`, and `flex-basis`.

```css
/* Longhand */
.item {
    flex-grow: 1;
    flex-shrink: 1;
    flex-basis: 0;
}

/* Shorthand */
.item {
    flex: 1 1 0;
    /* grow shrink basis */
}
```

**Common Shorthand Values:**

```css
/* flex: 1 - Most common! */
.item {
    flex: 1;
    /* = flex: 1 1 0 */
    /* Can grow, can shrink, no base size */
    /* USE THIS for equal-sized items */
}

/* flex: auto */
.item {
    flex: auto;
    /* = flex: 1 1 auto */
    /* Can grow, can shrink, starts at content size */
}

/* flex: none */
.item {
    flex: none;
    /* = flex: 0 0 auto */
    /* Can't grow, can't shrink, fixed size */
}

/* flex: initial (default) */
.item {
    flex: initial;
    /* = flex: 0 1 auto */
    /* Can't grow, can shrink, starts at content size */
}

/* Custom values */
.item {
    flex: 2;           /* = flex: 2 1 0 */
    flex: 200px;       /* = flex: 1 1 200px */
    flex: 2 2 10%;     /* = flex: 2 2 10% */
}
```

**CRITICAL INTERVIEW TIP:** Always use shorthand! It has better defaults.

```css
/* ❌ DON'T: Longhand can have unexpected behavior */
.item {
    flex-grow: 1;  /* Other values not set properly */
}

/* ✅ DO: Shorthand sets sensible defaults */
.item {
    flex: 1;  /* Clear and predictable */
}
```

**Common Patterns:**
```css
/* Equal-width columns */
.column { flex: 1; }

/* Sidebar + main content */
.sidebar { flex: 0 0 250px; }  /* Fixed 250px */
.main { flex: 1; }              /* Take remaining */

/* Button group */
.btn { flex: 1; }  /* All buttons equal width */

/* Proportional sizing */
.small { flex: 1; }
.large { flex: 2; }  /* 2x larger */
```

### align-self

Overrides `align-items` for individual item (cross axis alignment).

```css
.item {
    align-self: auto;       /* Default: inherit from parent */
    align-self: flex-start;
    align-self: flex-end;
    align-self: center;
    align-self: baseline;
    align-self: stretch;
}
```

**Example:**
```css
.container {
    display: flex;
    align-items: center;  /* All items centered */
}

.special-item {
    align-self: flex-start;  /* This one at the start */
}
```

**Visual:**
```
┌─────────────────────────────────────┐
│ ┌──┐                                │  ← Item 1 (align-self: flex-start)
│ │1 │                                │
│ └──┘ ┌──┐ ┌──┐                     │  ← Items 2 & 3 (align-items: center)
│      │2 │ │3 │                     │
│      └──┘ └──┘                     │
└─────────────────────────────────────┘
```

**Use Cases:**
```css
/* Logo at top, other items centered */
.header {
    display: flex;
    align-items: center;
}
.logo {
    align-self: flex-start;
}

/* Form with special button alignment */
.form {
    display: flex;
    align-items: baseline;
}
.submit-btn {
    align-self: flex-end;
}
```

### order

Changes the **visual order** of items (doesn't affect DOM order).

```css
.item {
    order: 0;   /* Default */
    order: 1;   /* Appears later */
    order: -1;  /* Appears earlier */
}
```

**How it works:**
- Items sorted by `order` value (lowest first)
- Same `order` = DOM order
- Default is 0

**Example:**
```html
<div class="container">
    <div class="item1">1</div>
    <div class="item2">2</div>
    <div class="item3">3</div>
</div>
```

```css
.container { display: flex; }
.item1 { order: 2; }  /* Appears last */
.item2 { order: 1; }  /* Appears second */
.item3 { order: 0; }  /* Appears first (default) */

/* Visual result: 3, 2, 1 */
```

**Visual:**
```
DOM order:   [1] [2] [3]
Visual order: [3] [2] [1]
```

**Common Use Cases:**
```css
/* Move "New" badge to front */
.badge-new { order: -1; }

/* Reorder for mobile */
@media (max-width: 768px) {
    .main-content { order: 2; }
    .sidebar { order: 1; }  /* Sidebar first on mobile */
}

/* Featured item first */
.featured { order: -1; }
```

**Accessibility Warning:** `order` changes visual order but NOT:
- Tab order (keyboard navigation)
- Screen reader order

Use carefully or adjust DOM order instead!

## Common Flexbox Patterns

### 1. Perfect Centering (Holy Grail!)

```css
.container {
    display: flex;
    justify-content: center;  /* Horizontal center */
    align-items: center;      /* Vertical center */
    min-height: 100vh;        /* Full viewport height */
}
```

### 2. Equal-Height Columns

```css
.container {
    display: flex;
}
.column {
    flex: 1;  /* Equal width & height */
}
/* All columns automatically same height! */
```

### 3. Sticky Footer

```css
body {
    display: flex;
    flex-direction: column;
    min-height: 100vh;
}
.content {
    flex: 1;  /* Takes remaining space */
}
.footer {
    /* Stays at bottom */
}
```

### 4. Holy Grail Layout

```css
.page {
    display: flex;
    flex-direction: column;
    min-height: 100vh;
}
.main {
    display: flex;
    flex: 1;
}
.sidebar-left { flex: 0 0 200px; }
.content { flex: 1; }
.sidebar-right { flex: 0 0 200px; }
```

### 5. Responsive Navigation

```css
.nav {
    display: flex;
    justify-content: space-between;
}

@media (max-width: 768px) {
    .nav {
        flex-direction: column;
    }
}
```

### 6. Media Object

```css
.media {
    display: flex;
    align-items: flex-start;
}
.media-image {
    flex: 0 0 auto;
    margin-right: 20px;
}
.media-content {
    flex: 1;
}
```

### 7. Card Footer at Bottom

```css
.card {
    display: flex;
    flex-direction: column;
}
.card-content {
    flex: 1;  /* Pushes footer to bottom */
}
.card-footer {
    /* Always at bottom */
}
```

## Flexbox vs Grid - When to Use?

**Use Flexbox when:**
- One-dimensional layout (row OR column)
- Content-driven sizing (items based on content)
- Small-scale layouts (navigation, cards)
- You need items to wrap
- Alignment is primary concern

**Use Grid when:**
- Two-dimensional layout (rows AND columns)
- Layout-driven sizing (explicit grid structure)
- Large-scale layouts (page structure)
- Complex alignment requirements
- Overlapping elements

**Often use together:**
```css
.page {
    display: grid;              /* Overall layout */
    grid-template-columns: 250px 1fr;
}

.nav {
    display: flex;              /* Navigation items */
    justify-content: space-between;
}

.card-content {
    display: flex;              /* Card internals */
    flex-direction: column;
}
```

## Interview Key Points

### Most Common Questions:

**1. "Explain main axis vs cross axis"**
- **Main axis**: Direction set by `flex-direction` (row = horizontal, column = vertical)
- **Cross axis**: Perpendicular to main axis
- `justify-content` = main axis, `align-items` = cross axis

**2. "What does flex: 1 mean?"**
- `flex: 1` = `flex: 1 1 0`
- `flex-grow: 1` (can grow)
- `flex-shrink: 1` (can shrink)  
- `flex-basis: 0` (no base size)
- Result: Items are truly equal size

**3. "Difference between align-items and align-content?"**
- `align-items`: Aligns items within each flex line
- `align-content`: Aligns flex lines themselves (only with wrap)

**4. "How to center a div with flexbox?"**
```css
.container {
    display: flex;
    justify-content: center;
    align-items: center;
}
```

**5. "Explain flex-basis vs width"**
- `flex-basis`: Initial size in flex context, more flexible
- `width`: Fixed size, overridden by flex-basis
- Use `flex-basis` for flex items

**6. "Why use flex: 1 instead of flex-grow: 1?"**
- Shorthand sets all three properties with good defaults
- `flex-grow: 1` alone doesn't set shrink or basis
- Shorthand is clearer and more predictable

## Best Practices

```css
/* 1. Always use shorthand */
.item { flex: 1; }  /* ✅ Good */
.item { flex-grow: 1; }  /* ❌ Incomplete */

/* 2. Use gap instead of margin */
.container {
    display: flex;
    gap: 20px;  /* Modern, cleaner */
}

/* 3. Don't fight the defaults */
.container {
    display: flex;  /* flex-direction: row is default */
}

/* 4. Use min-width/max-width for constraints */
.item {
    flex: 1;
    min-width: 200px;  /* Won't shrink below 200px */
    max-width: 500px;  /* Won't grow above 500px */
}

/* 5. Consider wrapping for responsiveness */
.container {
    display: flex;
    flex-wrap: wrap;  /* Natural responsiveness */
}
```
