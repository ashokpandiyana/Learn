## Chapter 1: CSS Fundamentals

### 1.1 Syntax and Selectors
- Basic syntax (selector, property, value)
- Type selectors, class selectors, ID selectors
- Universal selector (`*`)
- Attribute selectors (`[attr]`, `[attr=value]`, `[attr^=value]`)
- **Important:** Specificity hierarchy (inline > ID > class > element)

### 1.2 The Cascade
- Cascade algorithm (Origin, Specificity, Order)
- Inheritance and `inherit` keyword
- `initial`, `unset`, `revert` values
- **Note:** Understanding cascade is crucial for debugging

### 1.3 Box Model
- Content, padding, border, margin
- `box-sizing: border-box` vs `content-box`
- Collapsing margins
- **Production Tip:** Always use `box-sizing: border-box` globally

### 1.4 CSS Values and Units
- Absolute units (px, pt, cm)
- Relative units (em, rem, %, vw, vh, vmin, vmax)
- `ch`, `ex` units
- **Interview Focus:** Explain em vs rem clearly

## Chapter 5: Flexbox (Critical for Interviews)

### 5.1 Flex Container Properties

- `flex-wrap` (nowrap, wrap, wrap-reverse)

- `align-self`
- `order`
