4. Progressive Enhancement Strategy

Feature Detection with @supports
/* Base styles (work everywhere) */ .container { display: block; } /* Enhanced for flexbox support */ @supports (display: flex) { .container { display: flex; gap: 20px; } } /* Further enhanced for grid support */ @supports (display: grid) { .container { display: grid; grid-template-columns: repeat(auto-fit, minmax(250px, 1fr)); } }

5. Common Browser-Specific Fixes

Safari Flexbox Fix
/* Safari flex shrinking bug */ .flex-item { flex: 1; min-width: 0; /* Safari fix */ }
iOS Input Zoom Prevention
/* Prevent zoom on focus in iOS */ input, select, textarea { font-size: 16px; /* Must be >= 16px */ }
IE11 Flexbox Fix
/* IE11 flex bug - needs all 3 values */ .flex-item { flex: 1 1 auto; /* NOT just: flex: 1 */ } /* IE11 min-height flex bug */ .flex-container { display: flex; flex-direction: column; } .flex-item { min-height: 1px; /* IE11 fix */ }
Remove Default Styling (Cross-Browser)
/* Remove default appearance */ input, button, select { -webkit-appearance: none; -moz-appearance: none; appearance: none; } /* iOS tap highlight */ button, a { -webkit-tap-highlight-color: rgba(0,0,0,0); }

6. Autoprefixer (Recommended Tool)

üí° Don't manually add prefixes! Use Autoprefixer to automatically add vendor prefixes based on your browser support targets.
What You Write:
.element { display: flex; transform: rotate(45deg); transition: all 0.3s ease; }
What Autoprefixer Outputs:
.element { display: -webkit-box; display: -ms-flexbox; display: flex; -webkit-transform: rotate(45deg); -ms-transform: rotate(45deg); transform: rotate(45deg); -webkit-transition: all 0.3s ease; transition: all 0.3s ease; }
Setup (PostCSS)
// package.json { "browserslist": [ "last 2 versions", "> 1%", "not dead" ] } // postcss.config.js module.exports = { plugins: [ require('autoprefixer') ] }
