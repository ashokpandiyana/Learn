# Chapter 11: Effects & Filters

## 11.1 Box Shadow

Add depth and elevation to elements with shadows.

### Shadow Scale

```html
<!-- Shadow utilities from subtle to dramatic -->
<div class="shadow-sm">Subtle shadow</div>
<div class="shadow">Default shadow</div>
<div class="shadow-md">Medium shadow</div>
<div class="shadow-lg">Large shadow</div>
<div class="shadow-xl">Extra large shadow</div>
<div class="shadow-2xl">2XL shadow (most dramatic)</div>

<!-- Special shadows -->
<div class="shadow-inner">Inset shadow (pressed effect)</div>
<div class="shadow-none">No shadow</div>
```

```html
<!-- Shadow with color -->
<div class="shadow-lg shadow-blue-500/50">Blue glow</div>
<div class="shadow-lg shadow-red-500/40">Red glow</div>
<div class="shadow-xl shadow-purple-500/30">Purple glow</div>

<div class="shadow-lg shadow-cyan-500/50 bg-cyan-500">
  Neon cyan
</div>

<!-- Brand colored shadow -->
<button class="bg-blue-500 text-white shadow-lg shadow-blue-500/50 
               hover:shadow-blue-500/70 transition-shadow">
  Glowing Button
</button>
```

### Custom Shadows in Config

```javascript
// tailwind.config.js
module.exports = {
  theme: {
    extend: {

        'button': '0 4px 14px rgba(0, 118, 255, 0.39)',
        'glow': '0 0 20px rgba(59, 130, 246, 0.5)',
        'inner-lg': 'inset 0 4px 6px rgba(0, 0, 0, 0.1)',
        '3xl': '0 35px 60px -15px rgba(0, 0, 0, 0.3)',
      },
    },
  },
}
```

```html
  Custom inset shadow
</div>

  Custom glow effect
</div>
```

### Opacity vs Color Opacity

```html
<!-- Element opacity (affects entire element + children) -->
<div class="opacity-50 bg-blue-500 text-white">
  Everything is 50% transparent
</div>

<!-- Color opacity (affects only that color) -->
<div class="bg-blue-500/50 text-white">
  Only background is 50% transparent, text is solid
</div>
```

### Common Opacity Patterns

```html
<!-- Disabled state -->
<button class="opacity-50 cursor-not-allowed" disabled>
  Disabled Button
</button>

<!-- Overlay -->
<div class="fixed inset-0 bg-black opacity-50"></div>

<!-- Hover fade -->
<div class="opacity-75 hover:opacity-100 transition-opacity">
  Hover to reveal
</div>

<!-- Loading skeleton -->
<div class="animate-pulse opacity-75">Loading...</div>
```

## 11.3 Filters

Apply visual effects to elements.

### Blur

```html
<!-- Blur scale -->
<div class="blur-none">No blur</div>
<div class="blur-sm">Small blur (4px)</div>
<div class="blur">Default blur (8px)</div>

<div class="blur-lg">Large blur (16px)</div>
<div class="blur-xl">XL blur (24px)</div>
<div class="blur-2xl">2XL blur (40px)</div>
<div class="blur-3xl">3XL blur (64px)</div>

<!-- Arbitrary blur -->
<div class="blur-[2px]">Custom 2px blur</div>
```

### Brightness

```html
<!-- Brightness control -->
```

### Contrast

```html
<img class="contrast-0" src="...">    <!-- No contrast (gray) -->
<img class="contrast-50" src="...">   <!-- Low contrast -->
<img class="contrast-75" src="...">   <!-- Reduced contrast -->
<img class="contrast-100" src="...">  <!-- Normal -->
```

### Drop Shadow

Unlike box-shadow, drop-shadow follows the shape (great for PNGs).

```html
<img class="drop-shadow-sm" src="...">   <!-- Subtle -->
<img class="drop-shadow" src="...">      <!-- Default -->
<img class="drop-shadow-md" src="...">   <!-- Medium -->
<img class="drop-shadow-lg" src="...">   <!-- Large -->
<img class="drop-shadow-xl" src="...">   <!-- Extra large -->
<img class="drop-shadow-2xl" src="...">  <!-- 2XL -->
<img class="drop-shadow-none" src="..."> <!-- No shadow -->
```

## 11.6 Background Blend Mode

## ğŸ’¡ Interview Quick Tips

1. **"box-shadow vs drop-shadow?"**
   - `box-shadow`: Rectangle around element box
   - `drop-shadow`: Follows actual shape (PNG transparency)

2. **"How to create glass morphism?"**
   - `backdrop-blur-lg bg-white/10 border border-white/20`

3. **"Filter vs backdrop-filter?"**
   - `filter`: Affects the element itself
   - `backdrop-filter`: Affects what's behind the element

4. **"How to add colored shadows?"**
   - `shadow-lg shadow-blue-500/50`

5. **"Opacity on element vs color?"**
   - `opacity-50`: Entire element + children transparent
   - `bg-blue-500/50`: Only that specific color transparent
