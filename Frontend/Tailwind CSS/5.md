## 5.2 Color Utilities

### Text Color

```html
<p class="text-black">Pure black</p>
<p class="text-white">Pure white</p>
<p class="text-transparent">Transparent text</p>
<p class="text-current">Inherits current color</p>

<!-- With color palette -->
<p class="text-gray-500">Gray 500</p>
<p class="text-blue-600">Blue 600</p>
<p class="text-red-700">Red 700</p>
```

### Background Color

```html
<div class="bg-white">White background</div>
<div class="bg-black">Black background</div>
<div class="bg-transparent">Transparent background</div>

<!-- With color palette -->
<div class="bg-gray-100">Light gray background</div>
<div class="bg-blue-500">Blue background</div>
<div class="bg-emerald-50">Very light emerald</div>
```

### Border Color

```html
<div class="border border-gray-200">Light border</div>
<div class="border border-blue-500">Blue border</div>
<div class="border-t-4 border-t-red-500">Red top border</div>

<!-- Individual sides -->
<div class="border-l-4 border-l-green-500">Left border</div>
<div class="border-x border-x-gray-300">Horizontal borders</div>
```

### Ring Color (Focus Rings)

```html
<!-- Ring = box-shadow based outline -->
<button class="ring-2 ring-blue-500">Blue ring</button>
<input class="focus:ring-2 focus:ring-blue-500 focus:ring-offset-2">

<!-- Ring offset (space between ring and element) -->
<button class="ring-2 ring-blue-500 ring-offset-2 ring-offset-white">
  With offset
</button>
```

### Divide Color (Borders Between Children)

```html
<div class="divide-y divide-gray-200">
  <div class="py-4">Item 1</div>
  <div class="py-4">Item 2</div>  <!-- border-top here -->
  <div class="py-4">Item 3</div>  <!-- border-top here -->
</div>
```

### Outline Color

```html
<button class="outline outline-2 outline-blue-500">
  Outlined button
</button>
```

### Accent Color (Form Controls)

```html
<!-- Colors native form controls -->
<input type="checkbox" class="accent-pink-500">
<input type="radio" class="accent-emerald-500">
<input type="range" class="accent-blue-500">
<progress class="accent-purple-500" value="70" max="100"></progress>
```

### Caret Color (Text Cursor)

```html
<input type="text" class="caret-pink-500" placeholder="Pink cursor">
<textarea class="caret-blue-500">Blue text cursor</textarea>
```

### SVG Colors

```html
<svg class="fill-blue-500 stroke-blue-700">
  <!-- fill controls the inside color -->
  <!-- stroke controls the outline color -->
</svg>

<!-- Stroke width -->
<svg class="stroke-2 stroke-gray-500">...</svg>
```

## 5.3 Opacity Modifiers

Tailwind allows adding opacity to any color using the slash syntax.

### Syntax: `{utility}-{color}-{shade}/{opacity}`

```html
<!-- Background opacity -->
<div class="bg-black/50">50% black (semi-transparent)</div>
<div class="bg-blue-500/75">75% blue</div>
<div class="bg-white/10">10% white (glass effect)</div>

<!-- Text opacity -->
<p class="text-gray-900/80">80% opacity text</p>

<!-- Border opacity -->
<div class="border border-black/20">20% black border</div>

<!-- Ring opacity -->
<button class="ring-2 ring-blue-500/50">50% opacity ring</button>
```

### Available Opacity Values

```
/0   - 0%   (transparent)
/5   - 5%
/10  - 10%
/15  - 15%
/20  - 20%
/25  - 25%
/30  - 30%
/35  - 35%
/40  - 40%
/45  - 45%
/50  - 50%
/55  - 55%
/60  - 60%
/65  - 65%
/70  - 70%
/75  - 75%
/80  - 80%
/85  - 85%

/95  - 95%
```

### Arbitrary Opacity

```html
<div class="bg-blue-500/[.06]">6% opacity</div>
<div class="bg-red-500/[0.15]">15% opacity</div>
```

## 5.4 Current Color & Inherit

```html
<!-- current = uses the element's text color -->
<div class="text-blue-500">
  <svg class="fill-current">Uses blue-500</svg>
  <span class="border-current">Blue border</span>
</div>

<!-- inherit = inherits from parent -->
<div class="text-red-500">
  <span class="text-inherit">Also red</span>
</div>

<!-- Useful for icons that should match text -->
<button class="text-gray-700 hover:text-blue-600">
  <svg class="fill-current w-5 h-5">...</svg>
  Click me
</button>
```

## ğŸ’¡ Interview Quick Tips

1. **"How many gray palettes does Tailwind have?"**
   - 5: slate, gray, zinc, neutral, stone

2. **"How to add opacity to colors?"**
   - Slash syntax: `bg-blue-500/50`

3. **"What's the difference between ring and outline?"**
   - Ring: box-shadow based (doesn't affect layout)
   - Outline: CSS outline (doesn't affect layout)
   - Border: Actual border (affects layout)

4. **"How to make gradient text?"**
   - `bg-gradient-to-r from-X to-Y bg-clip-text text-transparent`

5. **"How to support CSS variable colors with opacity?"**
   - Use `rgb(var(--color) / <alpha-value>)` in config
