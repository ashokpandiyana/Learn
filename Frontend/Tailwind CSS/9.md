# Chapter 9: Backgrounds

## 9.1 Background Color

Apply solid background colors to elements.

```html
<!-- Basic colors -->
<div class="bg-white">White background</div>
<div class="bg-black">Black background</div>
<div class="bg-transparent">Transparent background</div>

<!-- Color palette -->
<div class="bg-gray-100">Light gray</div>
<div class="bg-blue-500">Blue 500</div>
<div class="bg-red-600">Red 600</div>
<div class="bg-emerald-400">Emerald 400</div>

<!-- With opacity -->
<div class="bg-black/50">50% black</div>
<div class="bg-blue-500/75">75% blue</div>
<div class="bg-white/10">10% white (glass effect)</div>

<!-- Arbitrary colors -->
<div class="bg-[#1da1f2]">Twitter blue</div>
<div class="bg-[rgb(255,100,50)]">RGB color</div>
<div class="bg-[hsl(200,100%,50%)]">HSL color</div>
```

## 9.3 Background Size

Control how background images are sized.

```html
<!-- Auto: Original size -->
<div class="bg-auto" style="background-image: url(...)">
  Background at natural size
</div>

<!-- Cover: Scale to cover, maintain ratio, may crop -->
<div class="bg-cover" style="background-image: url(...)">
  Covers entire container
</div>

<!-- Contain: Scale to fit, maintain ratio, may have gaps -->
<div class="bg-contain" style="background-image: url(...)">
  Fully visible, may have empty space
</div>

<!-- Arbitrary values -->
<div class="bg-[length:200px_100px]">Fixed dimensions</div>
<div class="bg-[length:50%_auto]">50% width, auto height</div>
```

## 9.5 Background Repeat

Control how background images tile.

```html
<!-- Repeat (default) -->
<div class="bg-repeat">Tiles in both directions</div>

<!-- No repeat -->
<div class="bg-no-repeat">Single instance, no tiling</div>

<!-- Repeat on one axis -->
<div class="bg-repeat-x">Tiles horizontally only</div>
<div class="bg-repeat-y">Tiles vertically only</div>

<!-- Round: Scale to fit without clipping -->
<div class="bg-repeat-round">
  Scales images to fit whole number of tiles
</div>

<!-- Space: Distribute with even spacing -->
<div class="bg-repeat-space">
  Tiles with even spacing, no clipping
</div>
```

## 9.6 Background Attachment

Control background scroll behavior.

```html
<!-- Scroll with content (default) -->
<div class="bg-scroll">
  Background scrolls with content
</div>

<!-- Fixed to viewport (parallax effect) -->
<div class="bg-fixed">
  Background stays fixed while content scrolls
</div>

<!-- Fixed to element's scroll container -->
<div class="bg-local">
  Background scrolls with element's content (in scrollable container)
</div>
```

### Parallax Effect Example

```html
<!-- Simple parallax section -->
<section class="relative h-screen bg-fixed bg-cover bg-center"
         style="background-image: url('hero.jpg')">
  <div class="absolute inset-0 bg-black/40 flex items-center justify-center">
    <h1 class="text-5xl font-bold text-white">Parallax Hero</h1>
  </div>
</section>

<section class="bg-white p-20">
  <p>Regular content section...</p>
</section>

<section class="h-screen bg-fixed bg-cover bg-center"
         style="background-image: url('another.jpg')">
  <!-- Another parallax section -->
</section>
```

## 9.7 Background Clip

Control how far the background extends.

```html
<!-- Clip to border edge (default) -->
<div class="bg-clip-border p-4 border-4 border-dashed border-blue-500 bg-blue-200">
  Background extends under border
</div>

<!-- Clip to padding edge -->
<div class="bg-clip-padding p-4 border-4 border-dashed border-blue-500 bg-blue-200">
  Background stops at padding
</div>

<!-- Clip to content edge -->
<div class="bg-clip-content p-4 border-4 border-dashed border-blue-500 bg-blue-200">
  Background only in content area
</div>

<!-- Clip to text (gradient text!) -->
<div class="bg-clip-text text-transparent bg-gradient-to-r from-purple-500 to-pink-500">
  Background clips to text shape
</div>
```

## 9.8 Background Origin

Control where background positioning starts.

```html
<!-- Border origin -->
<div class="bg-origin-border">
  Background position relative to border box
</div>

<!-- Padding origin (default) -->
<div class="bg-origin-padding">
  Background position relative to padding box
</div>

<!-- Content origin -->
<div class="bg-origin-content">
  Background position relative to content box
</div>
```

## 9.9 Background Image

Set background images using utilities.

```html
<!-- No background image -->
<div class="bg-none">Removes background image</div>

<!-- Arbitrary background images -->
<div class="bg-[url('/images/hero.jpg')]">
  Background image via URL
</div>

<!-- Multiple backgrounds -->
<div class="bg-[url('/overlay.png'),url('/bg.jpg')]">
  Layered backgrounds
</div>

<!-- Inline for dynamic images -->
<div class="bg-cover bg-center" style="background-image: url('dynamic.jpg')">
  Using style for dynamic images
</div>
```

### Background Image in Config

```javascript
// tailwind.config.js
module.exports = {
  theme: {
    extend: {
      backgroundImage: {
        'hero-pattern': "url('/images/hero-pattern.svg')",
        'footer-texture': "url('/images/footer-texture.png')",
        'gradient-radial': 'radial-gradient(var(--tw-gradient-stops))',
        'gradient-conic': 'conic-gradient(var(--tw-gradient-stops))',
      },
    },
  },
}
```

```html
<!-- Use configured images -->
<div class="bg-hero-pattern">Custom pattern</div>
<div class="bg-footer-texture">Footer background</div>
<div class="bg-gradient-radial from-blue-500 to-purple-600">Radial gradient</div>
```

## ğŸ’¡ Interview Quick Tips

1. **"How to create gradient text?"**
   - `bg-gradient-to-r from-X to-Y bg-clip-text text-transparent`

2. **"bg-cover vs bg-contain?"**
   - `bg-cover`: Fills container, may crop
   - `bg-contain`: Full image visible, may have gaps

3. **"What's bg-fixed used for?"**
   - Parallax effects
   - Background stays fixed while content scrolls

4. **"How to add overlay on image?"**
   - Absolute positioned div with `bg-black/50`
   - Or gradient: `bg-gradient-to-t from-black to-transparent`

5. **"How to make gradient border?"**
   - Wrapper with gradient bg + inner element with solid bg
   - Use padding on wrapper for border width
