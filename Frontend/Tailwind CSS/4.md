# Chapter 4: Typography

## 4.1 Font Family

Tailwind provides three default font stacks optimized for different purposes.

### Default Font Families

```html
<!-- Sans-serif (default for most UI) -->
<p class="font-sans">
  Uses: ui-sans-serif, system-ui, -apple-system, BlinkMacSystemFont, 
        "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif
</p>

<!-- Serif (for editorial/traditional content) -->
<p class="font-serif">
  Uses: ui-serif, Georgia, Cambria, "Times New Roman", Times, serif
</p>

<!-- Monospace (for code) -->
<p class="font-mono">
  Uses: ui-monospace, SFMono-Regular, Menlo, Monaco, Consolas, 
        "Liberation Mono", "Courier New", monospace
</p>
```

### Custom Font Configuration

```javascript
// tailwind.config.js
module.exports = {
  theme: {
    extend: {
      fontFamily: {
        // Add new font families
        sans: ['Inter', 'system-ui', 'sans-serif'],
        heading: ['Poppins', 'sans-serif'],
        body: ['Open Sans', 'sans-serif'],
        display: ['Playfair Display', 'serif'],
      },
    },
  },
}
```

```html
<!-- Usage -->
<h1 class="font-heading">Custom Heading Font</h1>
<p class="font-body">Custom body font</p>
```

### Loading Custom Fonts

```html
<!-- In your HTML head -->
<link rel="preconnect" href="https://fonts.googleapis.com">
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap" rel="stylesheet">

<!-- Or use @font-face in your CSS -->
<style>
@font-face {
  font-family: 'Inter';
  src: url('/fonts/Inter.woff2') format('woff2');
  font-weight: 400 700;
  font-display: swap;
}
</style>
```

## 4.4 Line Height

### Relative Line Heights

```html
<!-- Relative to font size -->
<p class="leading-none">line-height: 1 (no spacing)</p>
<p class="leading-tight">line-height: 1.25</p>
<p class="leading-snug">line-height: 1.375</p>
<p class="leading-normal">line-height: 1.5 (default)</p>
<p class="leading-relaxed">line-height: 1.625</p>
<p class="leading-loose">line-height: 2</p>
```

### Fixed Line Heights

```html
<!-- Fixed pixel values -->
<p class="leading-3">line-height: 12px (0.75rem)</p>
<p class="leading-4">line-height: 16px (1rem)</p>
<p class="leading-5">line-height: 20px (1.25rem)</p>
<p class="leading-6">line-height: 24px (1.5rem)</p>
<p class="leading-7">line-height: 28px (1.75rem)</p>
<p class="leading-8">line-height: 32px (2rem)</p>
<p class="leading-9">line-height: 36px (2.25rem)</p>
<p class="leading-10">line-height: 40px (2.5rem)</p>
```

### When to Use Each

```html
<!-- Headlines: Tight line height -->
<h1 class="text-5xl font-bold leading-tight">
  Big Bold Headlines<br>Need Tight Spacing
</h1>

<!-- Body text: Relaxed for readability -->
<p class="text-base leading-relaxed">
  Body text needs more breathing room for comfortable reading,
  especially for longer paragraphs like this one.
</p>

<!-- UI elements: Often use leading-none -->
<button class="text-sm leading-none py-2">
  Button Text
</button>
```

### Usage Guidelines

```html
<!-- Headlines: Slightly tighter -->
<h1 class="text-5xl font-bold tracking-tight">
  Headline with Tight Tracking
</h1>

<!-- ALL CAPS: Needs wider tracking -->
<p class="uppercase tracking-widest text-sm">
  SMALL CAPS TEXT
</p>

<!-- Body text: Normal tracking -->
<p class="tracking-normal">
  Regular body text looks best with default tracking.
</p>
```

## 4.9 Text Overflow & Wrapping

### Truncation (Single Line)

```html
<!-- truncate = overflow-hidden + text-ellipsis + whitespace-nowrap -->
<p class="truncate w-64">
  This very long text will be truncated with an ellipsis...
</p>

<!-- Individual utilities -->
<p class="overflow-hidden text-ellipsis whitespace-nowrap w-64">
  Same as truncate
</p>
```

### Multi-line Truncation (Line Clamp)

```html
<p class="line-clamp-1">One line then ellipsis...</p>
<p class="line-clamp-2">Two lines then ellipsis...</p>
<p class="line-clamp-3">Three lines then ellipsis...</p>
<p class="line-clamp-4">Four lines then ellipsis...</p>
<p class="line-clamp-5">Five lines then ellipsis...</p>
<p class="line-clamp-6">Six lines then ellipsis...</p>
<p class="line-clamp-none">Remove line clamp</p>
```

### Text Overflow Behavior

```html
<p class="text-ellipsis overflow-hidden">Ellipsis on overflow</p>
<p class="text-clip overflow-hidden">Clipped (no ellipsis)</p>
```

### Whitespace Handling

```html
<p class="whitespace-normal">Normal wrapping (default)</p>
<p class="whitespace-nowrap">No wrapping - stays on one line</p>
<p class="whitespace-pre">Preserves whitespace and newlines (like <pre>)</p>
<p class="whitespace-pre-line">Preserves newlines, collapses spaces</p>
<p class="whitespace-pre-wrap">Preserves whitespace, allows wrapping</p>
<p class="whitespace-break-spaces">Preserves all, breaks on spaces</p>
```

### Word Break

```html
<p class="break-normal">Default word breaking</p>
<p class="break-words">Break long words (overflow-wrap: break-word)</p>
<p class="break-all">Break anywhere (word-break: break-all)</p>
<p class="break-keep">Keep words together (CJK)</p>
```

1. **"What line-height does text-base include?"**
   - 1.5rem (24px) with 1rem (16px) font size

2. **"How to truncate text with ellipsis?"**
   - Single line: `truncate` class
   - Multi-line: `line-clamp-{n}` classes

   - `text-balance`: Even line lengths (headlines)
   - `text-pretty`: Avoids orphans (paragraphs)

4. **"When use tabular-nums?"**
   - Tables with numbers
   - Prices/currencies
   - Dates/times
   - Any data that needs to align

5. **"Default body text setup?"**
   - `text-base leading-relaxed text-gray-700`
