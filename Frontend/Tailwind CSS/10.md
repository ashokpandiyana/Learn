# Chapter 10: Borders & Outlines

## 10.1 Border Width

Control the thickness of borders.

### All Sides

```html
<!-- Default 1px border -->
<div class="border">1px border all sides</div>

<!-- Specific widths -->
<div class="border-0">No border (0px)</div>
<div class="border-2">2px border</div>
<div class="border-4">4px border</div>
<div class="border-8">8px border</div>

<!-- Arbitrary width -->
<div class="border-[3px]">3px border</div>
<div class="border-[0.5rem]">0.5rem border</div>
```

### Individual Sides

```html
<!-- Top -->
<div class="border-t">1px top border</div>
<div class="border-t-2">2px top border</div>
<div class="border-t-4">4px top border</div>

<!-- Right -->
<div class="border-r">1px right border</div>
<div class="border-r-2">2px right border</div>

<!-- Bottom -->
<div class="border-b">1px bottom border</div>
<div class="border-b-4">4px bottom border</div>

<!-- Left -->
<div class="border-l">1px left border</div>
<div class="border-l-8">8px left border</div>
```

### Axis-Based

```html
<!-- Horizontal (left + right) -->
<div class="border-x">Horizontal borders</div>
<div class="border-x-2">2px horizontal borders</div>

<!-- Vertical (top + bottom) -->
<div class="border-y">Vertical borders</div>
<div class="border-y-4">4px vertical borders</div>
```

### Logical Properties (RTL Support)

```html
<!-- Start (left in LTR, right in RTL) -->
<div class="border-s-4">Start border</div>

<!-- End (right in LTR, left in RTL) -->
<div class="border-e-4">End border</div>
```

## 10.2 Border Color

Set the color of borders.

```html
<!-- All sides -->
<div class="border border-gray-300">Gray border</div>
<div class="border border-blue-500">Blue border</div>
<div class="border border-red-600">Red border</div>

<!-- With opacity -->
<div class="border border-black/50">50% black border</div>
<div class="border border-blue-500/25">25% blue border</div>

<!-- Inherit / Current -->
<div class="border border-inherit">Inherits border color</div>
<div class="text-red-500 border border-current">Uses text color</div>

<!-- Transparent -->
<div class="border border-transparent">Invisible border (useful for hover)</div>
```

### Individual Side Colors

```html
<div class="border-4 border-gray-200 
            border-t-blue-500 
            border-r-green-500 
            border-b-yellow-500 
            border-l-red-500">
  Multi-colored borders
</div>

<!-- Logical properties -->
<div class="border-4 border-s-blue-500">Start color</div>
<div class="border-4 border-e-green-500">End color</div>
```

### Arbitrary Colors

```html
<div class="border border-[#1da1f2]">Twitter blue</div>
<div class="border border-[rgb(255,100,50)]">RGB</div>
<div class="border border-[hsl(200,100%,50%)]">HSL</div>
```

## 10.4 Border Radius

```html

<div class="rounded-sm">Small (0.125rem / 2px)</div>
<div class="rounded">Default (0.25rem / 4px)</div>
<div class="rounded-md">Medium (0.375rem / 6px)</div>
<div class="rounded-lg">Large (0.5rem / 8px)</div>
<div class="rounded-xl">Extra large (0.75rem / 12px)</div>
<div class="rounded-2xl">2x large (1rem / 16px)</div>
<div class="rounded-3xl">3x large (1.5rem / 24px)</div>
<div class="rounded-full">Full (9999px - pill/circle)</div>
```

## 10.5 Divide Width

Add borders between child elements.

```html
<!-- Vertical dividers (between horizontally stacked children) -->
<div class="flex divide-x">
  <div class="px-4">Item 1</div>
  <div class="px-4">Item 2</div>  <!-- border-left here -->
  <div class="px-4">Item 3</div>  <!-- border-left here -->
</div>

<!-- Horizontal dividers (between vertically stacked children) -->
<div class="divide-y">
  <div class="py-4">Item 1</div>
  <div class="py-4">Item 2</div>  <!-- border-top here -->
  <div class="py-4">Item 3</div>  <!-- border-top here -->
</div>

<!-- Width options -->
<div class="divide-y-0">No dividers</div>
<div class="divide-y">1px dividers (default)</div>
<div class="divide-y-2">2px dividers</div>
<div class="divide-y-4">4px dividers</div>
<div class="divide-y-8">8px dividers</div>
```

## 10.6 Outline

Outlines don't affect layout (unlike borders).

### Outline Width

```html
<div class="outline outline-0">No outline</div>
<div class="outline outline-1">1px outline</div>
<div class="outline outline-2">2px outline</div>
<div class="outline outline-4">4px outline</div>
<div class="outline outline-8">8px outline</div>

<!-- Arbitrary -->
<div class="outline outline-[3px]">3px outline</div>
```

### Outline Color

```html
<div class="outline outline-2 outline-blue-500">Blue outline</div>
<div class="outline outline-2 outline-red-600">Red outline</div>
<div class="outline outline-2 outline-gray-300">Gray outline</div>

<!-- With opacity -->
<div class="outline outline-2 outline-black/50">50% black</div>
```

### Outline Style

```html
<div class="outline outline-2 outline-dashed">Dashed</div>
<div class="outline outline-2 outline-dotted">Dotted</div>
<div class="outline outline-2 outline-double">Double</div>
<div class="outline-none">No outline</div>
```

### Outline Offset

Space between element edge and outline.

```html
<div class="outline outline-2 outline-blue-500 outline-offset-0">No offset</div>
<div class="outline outline-2 outline-blue-500 outline-offset-1">1px offset</div>
<div class="outline outline-2 outline-blue-500 outline-offset-2">2px offset</div>
<div class="outline outline-2 outline-blue-500 outline-offset-4">4px offset</div>
<div class="outline outline-2 outline-blue-500 outline-offset-8">8px offset</div>

<!-- Negative offset (inside element) -->
<div class="outline outline-2 outline-blue-500 -outline-offset-2">-2px offset</div>
```

## 10.7 Ring

Ring creates a box-shadow based outline (doesn't affect layout).

### Ring Width

```html
<div class="ring">3px ring (default)</div>
<div class="ring-0">No ring</div>
<div class="ring-1">1px ring</div>
<div class="ring-2">2px ring</div>
<div class="ring-4">4px ring</div>
<div class="ring-8">8px ring</div>

<!-- Inset ring (inside element) -->
<div class="ring-inset ring-2">Inset ring</div>
```

### Ring Color

```html
<div class="ring-2 ring-blue-500">Blue ring</div>
<div class="ring-2 ring-red-600">Red ring</div>
<div class="ring-2 ring-gray-300">Gray ring</div>

<!-- With opacity -->
<div class="ring-2 ring-blue-500/50">50% blue ring</div>
```

### Ring Offset

Space between ring and element.

```html
<div class="ring-2 ring-blue-500 ring-offset-0">No offset</div>
<div class="ring-2 ring-blue-500 ring-offset-1">1px offset</div>
<div class="ring-2 ring-blue-500 ring-offset-2">2px offset</div>
<div class="ring-2 ring-blue-500 ring-offset-4">4px offset</div>
<div class="ring-2 ring-blue-500 ring-offset-8">8px offset</div>

<!-- Ring offset color -->
<div class="ring-2 ring-blue-500 ring-offset-2 ring-offset-gray-100">
  Colored offset
</div>
```

## ğŸ’¡ Interview Quick Tips

1. **"Border vs Outline vs Ring?"**
   - `border`: Part of box model, affects layout
   - `outline`: Outside box, doesn't follow radius
   - `ring`: Box-shadow based, follows radius, best for focus

2. **"How to add dividers between list items?"**
   - Use `divide-y` on parent for vertical lists
   - Use `divide-x` on parent for horizontal lists

3. **"How to create rounded pill buttons?"**
   - Use `rounded-full` for full pill shape

4. **"Why use ring for focus states?"**
   - Follows border-radius
   - Has offset option with color
   - Doesn't affect layout

5. **"How to create gradient border?"**
   - Wrapper with gradient bg + padding
   - Inner element with solid bg
