## 17.2 Spacing Arbitrary Values

### Width & Height

```html
<!-- Exact pixel values -->
<div class="w-[137px]">137px width</div>
<div class="h-[283px]">283px height</div>
<div class="size-[50px]">50px Ã— 50px</div>

<!-- Percentages -->
<div class="w-[32.5%]">32.5% width</div>
<div class="h-[calc(100%-80px)]">100% minus 80px</div>

<!-- Viewport units -->
<div class="h-[100svh]">Small viewport height</div>
<div class="w-[50vw]">50% viewport width</div>

<!-- Container query units -->
<div class="w-[50cqw]">50% container width</div>

<!-- Other units -->
<div class="w-[10rem]">10rem</div>
<div class="w-[5em]">5em</div>
<div class="w-[100ch]">100 characters</div>
```

### Margin & Padding

```html
<!-- Exact values -->
<div class="p-[17px]">17px padding</div>
<div class="m-[2.5rem]">2.5rem margin</div>

<!-- Negative values -->
<div class="-mt-[13px]">-13px margin-top</div>
<div class="-mx-[5%]">-5% horizontal margin</div>

<!-- Complex values -->
<div class="p-[clamp(1rem,3vw,2rem)]">Fluid padding</div>
```

### Gap

```html
<div class="grid gap-[13px]">13px gap</div>
<div class="flex gap-x-[2.5rem] gap-y-[1rem]">Different axis gaps</div>
```

## 17.3 Color Arbitrary Values

```html
<!-- Hex colors -->
<div class="bg-[#1da1f2]">Hex 6-digit</div>
<div class="bg-[#f00]">Hex 3-digit</div>
<div class="bg-[#1da1f280]">Hex 8-digit (with alpha)</div>

<div class="bg-[rgba(29,161,242,0.5)]">RGBA</div>
<div class="bg-[rgb(29_161_242)]">RGB space syntax</div>
<div class="bg-[rgb(29_161_242/0.5)]">RGB with alpha</div>

<!-- HSL/HSLA -->
<div class="bg-[hsl(200,89%,53%)]">HSL</div>
<div class="bg-[hsla(200,89%,53%,0.5)]">HSLA</div>

<div class="bg-[hsl(200_89%_53%/0.5)]">HSL with alpha</div>

<div class="bg-[rebeccapurple]">Named color</div>

<div class="bg-[transparent]">Transparent</div>

<div class="bg-[color-mix(in_srgb,red,blue)]">Color mix</div>
```

## 17.6 CSS Functions in Arbitrary Values

### calc()

```html
<div class="w-[calc(100%-2rem)]">100% minus 2rem</div>
<div class="h-[calc(100vh-80px)]">Viewport minus header</div>
<div class="top-[calc(50%-100px)]">Centered with offset</div>
<div class="p-[calc(1rem+5px)]">Calculated padding</div>
```

### clamp()

```html
<!-- clamp(min, preferred, max) -->
<p class="text-[clamp(1rem,4vw,2rem)]">Fluid typography</p>
<div class="w-[clamp(200px,50%,600px)]">Fluid width</div>
<div class="p-[clamp(1rem,3vw,3rem)]">Fluid padding</div>
<div class="gap-[clamp(0.5rem,2vw,2rem)]">Fluid gap</div>
```

### min() and max()

```html
<div class="w-[min(100%,600px)]">Max 600px, responsive</div>
<div class="w-[max(300px,50%)]">At least 300px</div>
<div class="p-[min(5vw,3rem)]">Responsive with limit</div>
```

### var() - CSS Variables

```html
<div class="bg-[var(--brand-color)]">CSS variable</div>
<div class="w-[var(--sidebar-width)]">Variable width</div>
<div class="p-[var(--spacing-md,1rem)]">With fallback</div>

<!-- With calc -->
<div class="h-[calc(100vh-var(--header-height))]">
  Dynamic calculation
</div>
```

### url()

```html
<div class="bg-[url('/images/pattern.svg')]">Background image</div>
<div class="bg-[url('/hero.jpg')] bg-cover bg-center">Hero image</div>

<!-- Data URLs -->
<div class="bg-[url('data:image/svg+xml,...')]">Inline SVG</div>
```

## 17.7 Type Hints

When Tailwind can't determine the type of value, use type hints.

```html
<!-- Ambiguous: Is this a color or a length? -->
<div class="text-[var(--my-var)]">Ambiguous</div>

<!-- Type hints resolve ambiguity -->
<div class="text-[color:var(--text-color)]">This is a color</div>
<div class="text-[length:var(--font-size)]">This is a length</div>

<!-- Available type hints -->
color:       <!-- For color values -->
length:      <!-- For size/length values -->
percentage:  <!-- For percentage values -->
url:         <!-- For URLs -->
number:      <!-- For numbers -->
angle:       <!-- For angles (deg, rad) -->
line-width:  <!-- For border widths -->
```

### Common Type Hint Scenarios

```html
<!-- Background: color vs image -->
<div class="bg-[color:var(--bg)]">Color variable</div>
<div class="bg-[url:var(--bg-image)]">URL variable</div>

<!-- Font: family vs weight -->
<div class="font-[family-name:var(--font)]">Font family</div>
<div class="font-[number:var(--weight)]">Font weight</div>

<!-- Grid: number vs length -->
<div class="grid-cols-[number:var(--cols)]">Column count</div>
```

## 17.8 Arbitrary Properties

Apply ANY CSS property, even ones Tailwind doesn't support.

```html
<!-- Syntax: [property:value] -->

<!-- Not-commonly-used properties -->
<div class="[writing-mode:vertical-rl]">Vertical text</div>
<div class="[text-shadow:2px_2px_4px_#000]">Text shadow</div>
<div class="[hyphens:auto]">Auto hyphenation</div>
<div class="[scroll-snap-type:x_mandatory]">Snap scrolling</div>

<!-- Mask properties -->
<div class="[mask-image:linear-gradient(to_bottom,black,transparent)]">
  Gradient mask
</div>

<!-- Clip path -->
<div class="[clip-path:polygon(50%_0%,100%_100%,0%_100%)]">
  Triangle shape
</div>

<!-- CSS Grid specific -->
<div class="[grid-area:header]">Grid area</div>
<div class="[grid-template-areas:'a_a_b''c_d_d']">Template areas</div>

<!-- Accent and caret -->
<input class="[accent-color:#ff0000]">
<input class="[caret-color:blue]">

<!-- Perspective -->
<div class="[perspective:1000px]">3D perspective</div>
```

### Arbitrary Properties with Variants

```html
<!-- Works with all variants! -->
<div class="hover:[transform:scale(1.1)]">Hover scale</div>
<div class="dark:[color-scheme:dark]">Dark mode scheme</div>
<div class="md:[writing-mode:horizontal-tb]">Responsive writing mode</div>
<div class="group-hover:[clip-path:circle(100%)]">Group hover clip</div>
```

## 17.9 Arbitrary Variants

Create custom selectors for styling.

### Basic Syntax

```html
<!-- Syntax: [selector]:utility -->

<!-- Attribute selectors -->
<div class="[&[data-active]]:bg-blue-500">Active state</div>
<div class="[&[aria-expanded='true']]:rotate-180">Expanded rotation</div>

<!-- Pseudo-classes -->
<div class="[&:nth-child(3)]:bg-red-500">Third child</div>
<div class="[&:not(:last-child)]:border-b">All but last</div>
<div class="[&:first-of-type]:font-bold">First of type</div>

<!-- Descendant selectors -->
<div class="[&_p]:text-gray-600">All paragraph descendants</div>
<div class="[&>*]:border-b">Direct children</div>
<div class="[&_.active]:font-bold">Descendant with class</div>

<!-- Adjacent/sibling selectors -->
<div class="[&+&]:mt-4">Adjacent same element</div>
<div class="[&~p]:opacity-50">Following siblings</div>
```

### Parent Selectors

```html
<!-- Parent state affects child -->
<div class="[.dark_&]:text-white">White text when in .dark</div>
<div class="[.sidebar_&]:text-sm">Smaller in sidebar</div>
<div class="[[data-theme='blue']_&]:bg-blue-100">Theme-based</div>

<!-- Parent hover -->
<div class="[.card:hover_&]:opacity-100">Show on card hover</div>
```

### Complex Selectors

```html
<!-- Multiple conditions -->
<div class="[&:hover:not(:disabled)]:bg-blue-500">
  Hover but not disabled
</div>

<!-- Form validation -->
<input class="[&:user-invalid]:border-red-500">

<!-- Combined pseudo-elements and states -->
<div class="[&::before:hover]:opacity-100">Hover on ::before</div>

<!-- Media query-like -->
<div class="[@supports(display:grid)]:grid">
  Grid if supported
</div>

<div class="[@media(min-width:900px)]:flex">
  Custom breakpoint
</div>
```

## 17.10 CSS Variables Integration

### Defining Variables

```css
/* In your CSS */
:root {
  --brand-primary: #3b82f6;
  --brand-secondary: #8b5cf6;
  --spacing-page: 2rem;
  --header-height: 64px;
  --sidebar-width: 280px;
}

.dark {
  --brand-primary: #60a5fa;
  --brand-secondary: #a78bfa;
}
```

### Using in Tailwind

```html
<!-- Direct usage -->
<div class="bg-[var(--brand-primary)]">Brand color</div>
<div class="w-[var(--sidebar-width)]">Sidebar width</div>
<div class="h-[var(--header-height)]">Header height</div>
<div class="p-[var(--spacing-page)]">Page padding</div>

<!-- With calc -->
<main class="h-[calc(100vh-var(--header-height))]">
  Full height minus header
</main>

<!-- With fallbacks -->
<div class="p-[var(--custom-padding,1rem)]">
  With fallback
</div>
```

### Config Integration

```javascript
// tailwind.config.js
module.exports = {
  theme: {
    extend: {
      colors: {
        // CSS variable that supports opacity
        primary: 'rgb(var(--color-primary) / <alpha-value>)',
        // Simple CSS variable
        brand: 'var(--brand-color)',
      },
      spacing: {
        'header': 'var(--header-height)',
        'sidebar': 'var(--sidebar-width)',
      },
      fontSize: {
        'fluid': 'clamp(1rem, 2vw + 0.5rem, 1.5rem)',
      },
    },
  },
}
```

```html
<!-- Now use without brackets -->
<div class="bg-primary">Uses CSS variable</div>
<div class="bg-primary/50">With opacity!</div>
<div class="h-header">Header height</div>
<div class="text-fluid">Fluid text</div>
```

## ğŸ’¡ Interview Quick Tips

1. **"When to use arbitrary values?"**
   - One-off values not worth adding to config
   - Complex CSS functions (calc, clamp)
   - CSS variables integration

2. **"What are type hints?"**
   - Resolve ambiguity: `text-[color:var(--x)]`
   - Common: color, length, url, number

3. **"Arbitrary property syntax?"**
   - `[property:value]` for any CSS property
   - Works with variants: `hover:[text-shadow:...]`

4. **"Arbitrary variant syntax?"**
   - `[&selector]:utility` for custom selectors
   - `[.parent_&]:utility` for parent state

5. **"Should you overuse arbitrary values?"**
   - No! Add to config if used multiple times
   - Arbitrary = one-off exceptions
