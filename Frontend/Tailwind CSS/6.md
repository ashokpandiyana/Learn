## 6.2 Flex Container

### Enable Flexbox

```html
<!-- Block-level flex container -->
<div class="flex">
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
</div>

<!-- Inline flex container -->
<span class="inline-flex">
  <span>Item 1</span>
  <span>Item 2</span>
</span>
```

## 6.9 Flex Item Properties

### flex-1, flex-auto, flex-initial, flex-none

```html
<!-- flex-1: Grow and shrink, ignore initial size -->
<div class="flex">
  <div class="flex-1">Takes 1/3</div>
  <div class="flex-1">Takes 1/3</div>
  <div class="flex-1">Takes 1/3</div>
</div>
<!-- flex: 1 1 0% -->

<!-- flex-auto: Grow and shrink, consider initial size -->
<div class="flex">
  <div class="flex-auto">Wider content gets more space</div>
  <div class="flex-auto">Short</div>
</div>
<!-- flex: 1 1 auto -->

<!-- flex-initial: Shrink but don't grow (default) -->
<div class="flex">
  <div class="flex-initial">Will shrink if needed</div>
  <div class="flex-initial">But won't grow</div>
</div>
<!-- flex: 0 1 auto -->

<!-- flex-none: Don't grow or shrink -->
<div class="flex">
  <div class="flex-none w-32">Fixed 128px</div>
  <div class="flex-1">Takes remaining space</div>
</div>
<!-- flex: none -->
```

### Grow & Shrink

```html
<!-- Grow: Allow item to grow -->
<div class="flex">
  <div class="grow">Will grow to fill space</div>
  <div class="grow-0">Won't grow (default)</div>
</div>

<!-- Shrink: Allow item to shrink -->
<div class="flex">
  <div class="shrink">Will shrink if needed (default)</div>
  <div class="shrink-0">Won't shrink - keeps size</div>
</div>
```

### Practical grow/shrink example

```html
<!-- Fixed sidebar + fluid content -->
<div class="flex">
  <aside class="shrink-0 w-64">
    <!-- Won't shrink below 256px -->
    Sidebar
  </aside>
  <main class="grow">
    <!-- Takes all remaining space -->
    Content
  </main>
</div>
```

## 6.10 Order

Control visual order without changing HTML order.

```html
<div class="flex">
  <div class="order-3">1 (shows third)</div>
  <div class="order-1">2 (shows first)</div>
  <div class="order-2">3 (shows second)</div>
</div>

<!-- Special values -->
<div class="order-first">-9999 (always first)</div>
<div class="order-last">9999 (always last)</div>
<div class="order-none">0 (default position)</div>

<!-- Negative order -->
<div class="-order-1">-1 (before 0)</div>
```

## 6.11 Gap

Modern way to add space between flex items.

```html
<!-- Gap on both axes -->
<div class="flex gap-4">
  <div>1</div>
  <div>2</div>
  <div>3</div>
</div>

<!-- Different horizontal/vertical gaps -->
<div class="flex flex-wrap gap-x-4 gap-y-2">
  <!-- 16px horizontal, 8px vertical -->
</div>
```

### Gap vs Space

```html
<!-- gap (recommended) -->
<div class="flex gap-4">
  <!-- Works perfectly with flex-wrap -->
  <!-- No margin on first/last items -->
</div>

<!-- space-x (older approach) -->
<div class="flex space-x-4">
  <!-- Uses margin-left on all items except first -->
  <!-- Can cause issues with flex-wrap -->
</div>
```

## 6.12 Common Flexbox Patterns

### Center Everything

```html
<div class="flex items-center justify-center h-screen">
  <div>Perfectly centered!</div>
</div>
```

### Navigation Bar

```html
<nav class="flex items-center justify-between px-6 h-16 bg-white shadow">
  <!-- Logo -->
  <a class="text-xl font-bold">Logo</a>
  
  <!-- Nav Links (centered) -->
  <div class="flex items-center gap-6">
    <a class="hover:text-blue-500">Home</a>
    <a class="hover:text-blue-500">About</a>
    <a class="hover:text-blue-500">Services</a>
  </div>
  
  <!-- Actions -->
  <div class="flex items-center gap-4">
    <button>Login</button>
    <button class="px-4 py-2 bg-blue-500 text-white rounded">Sign Up</button>
  </div>
</nav>
```

### Card with Footer Stuck to Bottom

```html
<div class="flex flex-col h-64 bg-white rounded-lg shadow">
  <!-- Content (grows to fill) -->
  <div class="flex-1 p-4">
    <h3>Card Title</h3>
    <p>Variable amount of content...</p>
  </div>
  
  <!-- Footer (always at bottom) -->
  <div class="p-4 border-t">
    <button>Action</button>
  </div>
</div>
```

### Holy Grail Layout

```html
<div class="flex flex-col min-h-screen">
  <header class="h-16 bg-gray-100">Header</header>
  
  <div class="flex flex-1">
    <aside class="w-64 bg-gray-50">Sidebar</aside>
    <main class="flex-1 p-8">Main Content</main>
    <aside class="w-64 bg-gray-50">Right Sidebar</aside>
  </div>
  
  <footer class="h-20 bg-gray-100">Footer</footer>
</div>
```

### Input Group

```html
<div class="flex">
  <span class="flex items-center px-3 bg-gray-100 border border-r-0 rounded-l">
    @
  </span>
  <input class="flex-1 px-4 py-2 border rounded-r" placeholder="username">
</div>
```

### Equal Height Cards

```html
<div class="flex gap-4">
  <div class="flex-1 flex flex-col p-4 bg-white rounded shadow">
    <h3>Card 1</h3>
    <p class="flex-1">Short content</p>
    <button class="mt-4">Action</button>
  </div>
  
  <div class="flex-1 flex flex-col p-4 bg-white rounded shadow">
    <h3>Card 2</h3>
    <p class="flex-1">Much longer content that takes up more space...</p>
    <button class="mt-4">Action</button>
  </div>
</div>
```

## ğŸ’¡ Interview Quick Tips

1. **"Difference between justify-content and align-items?"**
   - `justify-content`: Main axis (row = horizontal)
   - `align-items`: Cross axis (row = vertical)

2. **"flex-1 vs flex-auto?"**
   - `flex-1`: Equal widths, ignores content size
   - `flex-auto`: Distributes space based on content

3. **"How to prevent flex item from shrinking?"**
   - Use `shrink-0` or `flex-none`

4. **"Gap vs space-x?"**
   - `gap`: Modern, works with wrap, no edge margins
   - `space-x`: Uses margins, issues with wrap

5. **"How to center a div with flexbox?"**
   - `flex items-center justify-center`
