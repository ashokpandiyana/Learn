## 7.2 Grid Container

### Enable Grid

```html
<!-- Block-level grid -->
<div class="grid">
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
</div>

<!-- Inline grid -->
<span class="inline-grid">
  <span>Item 1</span>
  <span>Item 2</span>
</span>
```

## 7.5 Grid Column Span

Make items span multiple columns.

```html
<!-- Span specific number of columns -->
<div class="col-span-1">Spans 1 column (default)</div>

<div class="col-span-3">Spans 3 columns</div>
```

## 7.9 Auto Columns/Rows

Size for implicitly created tracks (overflow items).

```html
<!-- Auto column sizing -->
<div class="grid grid-cols-3 auto-cols-auto">
  <!-- Extra columns: auto size -->
</div>
<div class="grid grid-cols-3 auto-cols-min">
  <!-- Extra columns: min-content -->
</div>
<div class="grid grid-cols-3 auto-cols-max">
  <!-- Extra columns: max-content -->
</div>
<div class="grid grid-cols-3 auto-cols-fr">
  <!-- Extra columns: 1fr (flexible) -->
</div>

<!-- Auto row sizing -->
<div class="grid grid-cols-3 auto-rows-auto">
  <!-- Extra rows: auto size (default) -->
</div>
<div class="grid grid-cols-3 auto-rows-min">
  <!-- Extra rows: min-content -->
</div>
<div class="grid grid-cols-3 auto-rows-max">
  <!-- Extra rows: max-content -->
</div>
<div class="grid grid-cols-3 auto-rows-fr">
  <!-- Extra rows: 1fr (equal height) -->
</div>

<!-- Arbitrary values -->
<div class="grid auto-rows-[200px]">
  <!-- All rows exactly 200px -->
</div>
<div class="grid auto-rows-[minmax(100px,auto)]">
  <!-- Min 100px, grow as needed -->
</div>
```

## 7.10 Grid Gap

Space between grid items.

```html
<!-- Gap on both axes -->
<div class="grid grid-cols-3 gap-4">16px gap everywhere</div>
<div class="grid grid-cols-3 gap-8">32px gap everywhere</div>

<!-- Different horizontal/vertical gaps -->
<div class="grid grid-cols-3 gap-x-4 gap-y-8">
  <!-- 16px column gap, 32px row gap -->
</div>

<!-- Arbitrary gap -->
<div class="grid gap-[30px]">Exactly 30px</div>
```

## 7.12 Subgrid

Inherit parent grid tracks (v3.4+).

```html
<div class="grid grid-cols-4 gap-4">
  <!-- Parent grid with 4 columns -->
  
  <div class="col-span-3 grid grid-cols-subgrid gap-4">
    <!-- Child uses parent's column tracks! -->
    <div>Aligns with parent col 1</div>
    <div>Aligns with parent col 2</div>
    <div>Aligns with parent col 3</div>
  </div>
  
  <div>Column 4</div>
</div>
```

## 7.13 Common Grid Patterns

```html
<!-- Auto-fit: Responsive without media queries! -->
<div class="grid grid-cols-[repeat(auto-fit,minmax(280px,1fr))] gap-6">
  <div class="p-6 bg-white rounded-lg shadow">Card 1</div>
  <div class="p-6 bg-white rounded-lg shadow">Card 2</div>
  <div class="p-6 bg-white rounded-lg shadow">Card 3</div>
  <div class="p-6 bg-white rounded-lg shadow">Card 4</div>
</div>
```

### Holy Grail Layout

```html
<div class="grid grid-cols-[200px_1fr_200px] grid-rows-[auto_1fr_auto] min-h-screen">
  <!-- Header: spans all columns -->
  <header class="col-span-3 p-4 bg-gray-100">Header</header>
  
  <!-- Left sidebar -->
  <aside class="p-4 bg-gray-50">Left</aside>
  
  <!-- Main content -->
  <main class="p-4">Main Content</main>
  
  <!-- Right sidebar -->
  <aside class="p-4 bg-gray-50">Right</aside>
  
  <!-- Footer: spans all columns -->
  <footer class="col-span-3 p-4 bg-gray-100">Footer</footer>
</div>
```

## ğŸ’¡ Interview Quick Tips

1. **"Grid vs Flexbox - when to use each?"**
   - Grid: 2D layouts, page structure, card grids
   - Flexbox: 1D layouts, nav bars, centering

2. **"How to create responsive grid without media queries?"**
   - `grid-cols-[repeat(auto-fit,minmax(250px,1fr))]`

3. **"auto-fit vs auto-fill?"**
   - `auto-fit`: Collapses empty tracks
   - `auto-fill`: Keeps empty tracks

4. **"What is subgrid?"**
   - Child grid inherits parent's track sizes
   - `grid-cols-subgrid` or `grid-rows-subgrid`

5. **"How to span full width in grid?"**
   - `col-span-full` or explicit `col-span-{n}`
