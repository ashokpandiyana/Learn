## 12.2 Scale

Resize elements larger or smaller.

### Scale Both Axes

```html
<!-- Scale uniformly -->
<div class="scale-0">0% (invisible)</div>
<div class="scale-50">50%</div>
<div class="scale-75">75%</div>
<div class="scale-90">90%</div>
<div class="scale-95">95%</div>
<div class="scale-100">100% (normal)</div>
<div class="scale-105">105%</div>
<div class="scale-110">110%</div>
<div class="scale-125">125%</div>
<div class="scale-150">150%</div>

<!-- Arbitrary values -->
<div class="scale-[1.75]">175%</div>
<div class="scale-[0.85]">85%</div>
```

### Scale Individual Axes

```html
<!-- Scale X (horizontal) -->
```

## 12.4 Translate

Move elements from their original position.

### Translate X (Horizontal)

```html
<!-- Using spacing scale -->
<div class="translate-x-0">No movement</div>
<div class="translate-x-1">Move 4px right</div>
<div class="translate-x-4">Move 16px right</div>
<div class="translate-x-8">Move 32px right</div>
<div class="translate-x-16">Move 64px right</div>

<!-- Negative (move left) -->
<div class="-translate-x-4">Move 16px left</div>
<div class="-translate-x-8">Move 32px left</div>

<!-- Percentage of element width -->
<div class="translate-x-1/2">Move 50% of own width</div>
<div class="translate-x-full">Move 100% of own width</div>
<div class="-translate-x-1/2">Move -50% of own width</div>
<div class="-translate-x-full">Move -100% of own width</div>
```

### Translate Y (Vertical)

```html
<!-- Using spacing scale -->
<div class="translate-y-0">No movement</div>
<div class="translate-y-4">Move 16px down</div>
<div class="translate-y-8">Move 32px down</div>

<!-- Negative (move up) -->
<div class="-translate-y-4">Move 16px up</div>
<div class="-translate-y-8">Move 32px up</div>

<!-- Percentage -->
<div class="translate-y-1/2">Move 50% of own height</div>
<div class="-translate-y-full">Move -100% of own height</div>
```

## 12.6 Transform Origin

Control the pivot point for transformations.

```html
<!-- Predefined origins -->
<div class="origin-center">Center (default)</div>
<div class="origin-top">Top center</div>
<div class="origin-top-right">Top right</div>
<div class="origin-right">Right center</div>
<div class="origin-bottom-right">Bottom right</div>
<div class="origin-bottom">Bottom center</div>
<div class="origin-bottom-left">Bottom left</div>
<div class="origin-left">Left center</div>
<div class="origin-top-left">Top left</div>

<!-- Arbitrary values -->
<div class="origin-[25%_75%]">Custom origin</div>
<div class="origin-[0_0]">Top-left corner</div>
```

## 12.8 Combining Transforms

Tailwind applies transforms in a specific order. Use multiple classes:

```html
<!-- Multiple transforms -->
<div class="scale-110 rotate-6 translate-x-4">
  Scaled, rotated, and moved
</div>

<!-- Complex hover effect -->
<div class="hover:scale-105 hover:-rotate-3 hover:-translate-y-2 
            transition-transform duration-300">
  Multi-transform hover
</div>

<!-- 3D effect -->
<div class="perspective-normal">
  <div class="rotate-y-12 rotate-x-6 scale-95">
    Tilted card
  </div>
</div>
```

## ğŸ’¡ Interview Quick Tips

1. **"Why use transforms for animations?"**
   - GPU accelerated (smooth 60fps)
   - Don't trigger layout recalculation
   - Only `transform` and `opacity` are fully optimized

2. **"How to center with transforms?"**
   - `absolute top-1/2 left-1/2 -translate-x-1/2 -translate-y-1/2`

3. **"transform-origin purpose?"**
   - Changes the pivot point for rotation/scale
   - Default is center

4. **"How to create flip cards?"**
   - Parent: `perspective-normal`
   - Container: `transform-3d`
   - Both sides: `backface-hidden`
   - Back: `rotate-y-180`

5. **"Scale vs width/height changes?"**
   - `scale`: GPU accelerated, doesn't affect layout
   - `width/height`: Triggers reflow, affects other elements
