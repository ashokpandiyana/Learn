## Z-Index

### Default Z-Index Scale

| Class | Value | Use Case |
|-------|-------|----------|
| `z-0` | 0 | Reset to default |
| `z-10` | 10 | Dropdowns, tooltips |
| `z-20` | 20 | Fixed headers |
| `z-30` | 30 | Sticky elements |
| `z-40` | 40 | Overlays |
| `z-50` | 50 | Modals, dialogs |
| `z-auto` | auto | Default stacking |
| `-z-10` | -10 | Behind content |

### Usage Examples

```html
<!-- Header -->
<header class="sticky top-0 z-50 bg-white">
  Sticky header
</header>

<!-- Dropdown -->
<div class="relative">
  <button>Menu</button>
  <div class="absolute z-10 mt-2 bg-white shadow-lg">
    Dropdown
  </div>
</div>

<!-- Modal -->
<div class="fixed inset-0 z-50 bg-black/50">
  <div class="bg-white p-6 rounded-lg">
    Modal content
  </div>
</div>

<!-- Background decoration -->
<div class="relative">
  <div class="-z-10 absolute inset-0 bg-gradient-to-r from-purple-500 to-pink-500 opacity-20">
    Background
  </div>
  <div class="relative z-0 p-8">
    Content
  </div>
</div>
```

### Z-Index Hierarchy

```
Typical stacking order:
├── Background elements: -z-10
├── Content: z-0 (default)
├── Dropdowns/tooltips: z-10
├── Sticky headers: z-20
├── Fixed navigation: z-30
├── Overlays: z-40
└── Modals: z-50
```

## Class Ordering Convention

### Standard Order

```
1. Layout
   position, display, flex/grid properties

2. Box Model (outside → in)
   width, height, margin, padding, border

3. Typography
   font-family, font-size, font-weight, text-align, color

4. Visual
   background, border-radius, shadow, opacity

5. Transforms & Effects
   transform, filter, transition, animation

6. Interactive States
   hover:, focus:, active:, disabled:

7. Responsive (smallest to largest)
   sm:, md:, lg:, xl:, 2xl:
```

### Before & After

```html
<!-- ❌ Random order -->
<div class="text-white hover:shadow-xl p-6 md:p-8 bg-blue-600 rounded-lg mb-4 flex sm:block shadow-md items-center">

<!-- ✅ Organized order -->
<div class="
  flex items-center
  mb-4 p-6
  bg-blue-600 text-white rounded-lg shadow-md
  hover:shadow-xl
  sm:block sm:p-8
">
```

## Summary

### Essential References

**Most Common Classes:**
- Layout: `flex`, `grid`, `block`
- Spacing: `p-4`, `m-4`, `gap-4`
- Sizing: `w-full`, `h-screen`
- Colors: `bg-white`, `text-gray-900`
- Typography: `text-base`, `font-medium`
- Borders: `rounded-lg`, `border`
- Shadows: `shadow-md`
- Interactive: `hover:`, `focus:`

**Key Patterns:**
- Center: `flex items-center justify-center`
- Responsive grid: `grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3`
- Sticky header: `sticky top-0 z-50`
- Card: `bg-white rounded-xl shadow-lg p-6`

### Quick Tips

1. **Mobile-first**: Base styles, then `sm:`, `md:`, etc.
2. **Class order**: Layout → Box Model → Typography → Visual → States → Responsive
3. **Focus states**: Always provide custom focus if removing outline
4. **Contrast**: Use gray-700+ on white backgrounds
5. **Semantic HTML**: Use proper elements with utilities
