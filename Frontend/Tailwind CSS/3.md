## 3.2 Margin Utilities

### Basic Margin Classes

```html
<!-- All sides -->
<div class="m-4">16px margin on all sides</div>

<!-- Horizontal (left + right) -->
<div class="mx-4">16px margin left and right</div>

<!-- Vertical (top + bottom) -->
<div class="my-4">16px margin top and bottom</div>

<!-- Individual sides -->
<div class="mt-4">margin-top: 16px</div>
<div class="mr-4">margin-right: 16px</div>
<div class="mb-4">margin-bottom: 16px</div>
<div class="ml-4">margin-left: 16px</div>
```

### Logical Properties (RTL Support)

```html
<!-- Logical properties adapt to text direction -->
<div class="ms-4">margin-inline-start: 16px</div>
<div class="me-4">margin-inline-end: 16px</div>

<!-- In LTR: ms = left, me = right -->
<!-- In RTL: ms = right, me = left -->
```

### Negative Margins

```html
<!-- Prefix with dash for negative values -->
<div class="-m-4">margin: -16px</div>
<div class="-mt-8">margin-top: -32px</div>
<div class="-mx-2">margin-left: -8px; margin-right: -8px</div>
```

### Auto Margins

```html
<!-- Center horizontally -->
<div class="mx-auto w-96">Centered container</div>

<!-- Push to right -->
<div class="ml-auto">Pushed to the right</div>

<!-- Flex item to right -->
<div class="flex">
  <span>Left</span>
  <span class="ml-auto">Right</span>
</div>
```

## 3.3 Padding Utilities

### Basic Padding Classes

```html
<!-- All sides -->
<div class="p-4">16px padding on all sides</div>

<!-- Horizontal (left + right) -->
<div class="px-4">16px padding left and right</div>

<!-- Vertical (top + bottom) -->
<div class="py-4">16px padding top and bottom</div>

<!-- Individual sides -->
<div class="pt-4">padding-top: 16px</div>
<div class="pr-4">padding-right: 16px</div>
<div class="pb-4">padding-bottom: 16px</div>
<div class="pl-4">padding-left: 16px</div>

<!-- Logical properties -->
<div class="ps-4">padding-inline-start: 16px</div>
<div class="pe-4">padding-inline-end: 16px</div>
```

### Common Padding Patterns

```html
<!-- Button padding -->
<button class="px-4 py-2">
  Horizontal 16px, Vertical 8px
</button>

<!-- Card padding -->
<div class="p-6">
  24px all around
</div>

<!-- Navigation item -->
<a class="px-3 py-2">
  Nav Link
</a>

<!-- Asymmetric padding -->
<div class="pt-8 pb-16 px-4">
  Different top/bottom
</div>
```

### Min-Height & Max-Height

```html
<!-- Min-height -->
<div class="min-h-0">min-height: 0</div>
<div class="min-h-full">min-height: 100%</div>
<div class="min-h-screen">min-height: 100vh</div>
<div class="min-h-svh">min-height: 100svh</div>

<div class="max-h-0">max-height: 0</div>
<div class="max-h-full">max-height: 100%</div>
<div class="max-h-screen">max-height: 100vh</div>
<div class="max-h-96">max-height: 384px</div>
```

## 3.6 Size Utility (Width + Height)

The `size-*` utility sets both width AND height simultaneously.

```html
<!-- Sets both width and height -->
<div class="size-4">width: 16px; height: 16px</div>
<div class="size-8">width: 32px; height: 32px</div>
<div class="size-16">width: 64px; height: 64px</div>
<div class="size-full">width: 100%; height: 100%</div>

<!-- Perfect for square elements -->
<img class="size-12 rounded-full" src="avatar.jpg" alt="Avatar">

<!-- Icons -->
<svg class="size-6">...</svg>

<!-- Equivalent to -->
<div class="w-4 h-4">Same as size-4</div>
```

## 3.7 Space Between (Gap Alternative)

The `space-x-*` and `space-y-*` utilities add margin between children.

```html
<!-- Horizontal space between children -->
<div class="flex space-x-4">
  <div>Item 1</div>  <!-- No margin-left -->
  <div>Item 2</div>  <!-- margin-left: 16px -->
  <div>Item 3</div>  <!-- margin-left: 16px -->
</div>

<!-- Vertical space between children -->
<div class="flex flex-col space-y-4">
  <div>Item 1</div>  <!-- No margin-top -->
  <div>Item 2</div>  <!-- margin-top: 16px -->
  <div>Item 3</div>  <!-- margin-top: 16px -->
</div>

<!-- Reverse space (for flex-row-reverse) -->
<div class="flex flex-row-reverse space-x-reverse space-x-4">
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
</div>
```

### Space vs Gap

```html
<!-- space-x (uses margins) -->
<div class="flex space-x-4">
  <!-- âš ï¸ Issues: Doesn't work with flex-wrap properly -->
</div>

<!-- gap (modern CSS gap) -->
<div class="flex gap-4">
  <!-- âœ… Better: Works with flex-wrap -->
</div>

<!-- When to use each: -->
<!-- gap: Flexbox/Grid layouts (preferred) -->
<!-- space: Simple lists, legacy browser support -->
```

## 3.8 Arbitrary Spacing Values

When you need exact values not in the scale:

```html
<!-- Arbitrary values with brackets -->
<div class="p-[17px]">Exactly 17px padding</div>
<div class="m-[2.5rem]">2.5rem margin</div>
<div class="w-[137px]">137px width</div>
<div class="h-[calc(100vh-80px)]">Calculated height</div>
<div class="gap-[clamp(1rem,5vw,3rem)]">Fluid gap</div>

<!-- Using CSS variables -->
<div class="p-[var(--custom-padding)]">CSS variable</div>

<!-- Negative arbitrary values -->
<div class="-mt-[17px]">Negative margin</div>
```

## ğŸ’¡ Interview Quick Tips

1. **"What's Tailwind's spacing base unit?"**
   - 4px (0.25rem)
   - `spacing-1` = 4px, `spacing-4` = 16px

2. **"Difference between space-x and gap?"**
   - `space-x`: Uses margins on children (except first)
   - `gap`: Modern CSS gap property (works better with wrapping)

3. **"h-screen vs h-dvh?"**
   - `h-screen`: 100vh (problematic on mobile)
   - `h-dvh`: 100dvh (adjusts for browser chrome)

4. **"What does size-4 do?"**
   - Sets both width and height to 1rem (16px)
   - Equivalent to `w-4 h-4`

5. **"How to center a div?"**
   - `mx-auto` for block element
   - `flex items-center justify-center` for flex
   - `grid place-items-center` for grid
